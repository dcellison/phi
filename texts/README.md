# The texts

This shelf holds Phi's translations and transmutations. A translation stays answerable to the source's claims and distinctions. A transmutation lets the same source change under Phi's five pillars and habits of thought. Phi prefers transmutation, while paired renderings keep the close analogue beside the Phi-shaped work so a reader can see what changed.

Short works each have one Markdown file. A book gets its own directory, where its chapter sequence and shared source witness stay together. The `sources/` directory holds source witnesses for the remaining works.
