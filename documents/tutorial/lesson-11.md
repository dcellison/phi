## Lesson 11: Building Complex Sentences

Welcome to the final stage of learning Phi's core grammar! In this lesson, we will learn how to create more complex sentences by embedding one clause inside another. This is called **subordination**.

A subordinate clause is a small sentence that acts as a descriptive part of a larger sentence. In Phi, true to its head-final nature, subordinate clauses always come **before** the main clause they modify.

### Temporal Clauses (when, before, after)

You can use a full sentence to describe the time when something happens. To do this, you form a complete clause and end it with a temporal postposition.

Let's learn two temporal postpositions:
*   `phoe`: before
*   `pheo`: after

To say, "Before I help, I think," we create the clause "I help" (`mia shele`) and add `phoe`. This whole phrase then comes before the main clause, "I think" (`mia nolu`).

*   **Phi:** `[mia shele] phoe mia nolu.`
*   **Gloss:** `[1SG help] BEFORE 1SG think.`
*   **Translation:** Before I help, I think.

Here's an example with `pheo`:

*   **Phi:** `[mia pholea] pheo mia hisu sela.`
*   **Gloss:** `[1SG create] AFTER 1SG feel peace.`
*   **Translation:** After I create, I feel peace.

### Purpose Clauses

You can also use a clause to explain the *purpose* of an action. To do this, you use the special purposive conjunction `lila`.

Unlike a postposition, `lila` is placed *after* the clause it modifies.

Let's break down an example: "We build mindfully in order to build a house."

*   **The Main Clause:** `weo nosa thekoa` (We build mindfully)
*   **The Purpose Clause:** `[hauno thekoa] lila` (in order to build a house)

Putting it all together in the correct `[Purpose Clause], [Main Clause]` order:

*   **Phi:** `[hauno thekoa] lila, weo nosa thekoa.`

This structure allows you to clearly state the "why" behind any action.

### Practice

*   How would you say, "I am speaking in order to help"?
*   How would you say, "They listen in order to understand love"?

### Key Takeaways

*   Use `[... V] lila` to create a clause that explains the purpose of an action.
*   The purpose clause comes before the main clause in the sentence.

### Lesson 11 Summary

*   A **subordinate clause** is a small sentence that describes a part of a larger one.
*   Subordinate clauses always come **before** the main clause.
*   Use `[clause] phoe` for "before" and `[clause] pheo` for "after."
*   Use `[... V] lila` to create a clause that explains the purpose of an action.
