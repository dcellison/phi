# Base vocabulary placement assessment

**Status:** Complete. Thirty former base roots now carry 35 module classifications, while 789 content roots remain in base vocabulary.

A learner who selects base vocabulary in the explorer should not meet a railway, a guild, a chisel, or a medicinal tonic on the way to ordinary Phi. Those words are still canonical and available to every speaker. Their module labels put them in the learning paths where their meanings do the most work.

## What placement means

The base filter is a learning choice, not a claim that its words are simpler in every sense. Before a speaker opens a domain module, Phi still needs broad language for bodies and agency, communication and place, materials and ordinary social life. A module classification removes no word from the language and changes no grammar, definition, example, or permission to use it.

This review is therefore narrower than a frequency list. A rare tree, instrument, emotion, or spiritual word does not enter the nearest module merely because a beginning learner may not know it. The word moves only when its lexical center belongs to an existing charter and the base layer can still explain the surrounding field without it.

## Decision rule

Each of the 819 content roots that began without a `modules` field was checked against five questions:

1. Does the entry's existing definition directly name a concept inside one or more current module charters?
2. Would a learner studying that field reasonably expect to find the word in its module index?
3. Is the word narrower than the base vocabulary needed to paraphrase that field?
4. Would module placement leave ordinary identity, consent, safety, testimony, agency, and communication available in base Phi?
5. Can the classification be added without changing the word's meaning?

A word had to pass all five. Multiple classifications were added only where the same definition has an independent home in each module. Use in another field was not enough by itself.

## Audit scope

The review covered every canonical file under `vocabulary/content/` and excluded `archive/`. It compared each unclassified root's word, gloss, concept, description, grammatical notes, profile use, and active corpus use with all eight module charters. Existing profile dependency tables supplied useful evidence, but their old placement labels were not treated as decisions that could never be revisited.

The first pass produced a deliberately broad candidate set. The preservation questions removed the technical words needed for paraphrase, along with ordinary actions and bodily self-report. They also removed broad social concepts and words whose several senses would make one module label misleading. The remaining thirty roots point to a specialized learning home without needing semantic repair.

Words for art, music, ritual, spiritual practice, and cultural life were not pushed into a current module merely to shorten the base list. A later profile may give some of them a genuine learning home. Until then, an inaccurate label would be tidier metadata and worse guidance.

## Completion and next work

The canonical entries carry the accepted classifications, the generated module index reflects all 352 memberships, and the profile counts distinguish roots coined in a module pass from older roots later classified there. The lexicon explorer already derives base vocabulary from the absence of a `modules` field, so it needs no special-case list.

The active-corpus revisit remains next. This assessment identifies older specialist words that appear in general teaching and literature, so the corpus review can ask whether each occurrence still reads naturally. Where it does not, the text can add a nearby explanation or use a base paraphrase. The words did not change. What changed is the promise about which ones a general learner is expected to study first.
