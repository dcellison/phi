# colours

> colours are usually used as adjectives to describe nouns, indicating the color of an object, person, or place. they are words that depict the hue, shade, or tint of something. colours can also be used as nouns. when used as nouns, they refer to the color itself or a variant of it.

colours in phi are based in nature, specifically in the colours of the rainbow. Below is a table of colour names in both phi and english. This list includes black, white, and grey to make the set of colours more practical.

| phi   | english |
| ----- | ------- |
| lawha | red     |
| sathe | orange  |
| phili | yellow  |
| hashe | green   |
| mipho | blue    |
| pewhe | indigo  |
| nisha | violet  |
| sathi | black   |
| thasi | white   |
| rawha | grey    |

## consider using the colour wheel

using the colours of the rainbow as colour names makes sense from the perspective of using nature in the language. but limiting the names of colours to just seven might be a bit restricting. if we use the standard colour wheel instead, we add five more colours. The colour wheel is just as "natural" as the rainbow, though perhaps a bit less romantic.

### primary colours

these colours stand alone and cannot be derived from other colours.

- red
- yellow
- blue

### secondary colours

these colours can be made by combining two primary colours. for example, mixing blue and yellow produces green.

- purple
- orange
- green

### tertiary colours

these colours are produced by combining a primary and a secondary colour. for example, mixing red and orange produces vermillion.

- vermillion
- amber
- chartreuse
- teal
- violet
- magenta
