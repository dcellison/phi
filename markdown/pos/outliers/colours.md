# colours

> colours are usually used as adjectives to describe nouns, indicating the color of an object, person, or place. they are words that depict the hue, shade, or tint of something. colours can also be used as nouns. when used as nouns, they refer to the color itself or a variant of it.

phi uses a set of base colours derived from the standard twelve-colour wheel, supplemented with black, white, and grey for practicality. the colour wheel itself is a natural representation of colour relationships.

### primary colours

these colours stand alone and cannot be derived from other colours by mixing.

- red
- yellow
- blue

### secondary colours

these colours can be made by combining two primary colours.

- purple (red + blue)
- orange (red + yellow)
- green (blue + yellow)

### tertiary colours

these colours are produced by combining a primary and an adjacent secondary colour.

- vermillion (red + orange)
- amber (yellow + orange)
- chartreuse (yellow + green)
- teal (blue + green)
- violet (blue + purple)
- magenta (red + purple)
