# Part 4: Reference by context

Shared attention can identify a separately presented artifact just as it identifies a nearby table, sign, or record. Demonstratives, ordinary nouns, and discourse context do that work without importing the source string.

Source specification outside Phi: `DO NOT MOVE UNTIL INSPECTED`

```
ha sirami ha kisholu phelu.
PROX record PROX specification hold.
(This record contains this specification.)
```

```
lo mia ha kisholu thekao.
PL 1SG PROX specification acknowledge.
(We acknowledge this specification.)
```

The Phi account can distinguish seeing the record, receiving notice, understanding it, accepting it, satisfying it, enforcing it, and deciding that it applies. Exact typography alone settles none of those relations. Most disputes about a posted notice are disputes about which of these happened, and Phi gives each one its own verb.
