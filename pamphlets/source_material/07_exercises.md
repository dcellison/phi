# Part 7: Exercises

## A. Choose the layer

For each item, choose canonical vocabulary, a transparent Phi expression, a vocabulary-module term, a productive onym, a translation, or separate source presentation: a person's preferred source-script name; a recurring word for system capacity; a one-time description of a broken wooden handle; a dosage; a legal clause; a community-approved name `samira`; and a technical identifier.

## B. Keep the artifact separate

Present the source record `temperature: 38.7 C` outside a Phi passage. Then write one Phi sentence saying that you read the record and another saying that you question the conclusion. Do not place the digits or unit inside either sentence.

## C. Test a name

Run `python3 scripts/validate_examples.py name samira`, then test `saweriko`, `mimimi`, and each retired boundary form. Explain whether rejection concerns word shape, duplicated syllables, or historical reservation.

## D. Separate source, translation, and analysis

Choose one short source sentence. Display it, give a labeled Phi translation, then add a Phi or surrounding-language note explaining one distinction the translation may not preserve.
