# Examples

Longer lines than the rest of this pamphlet, and more varied: particles stacked on the verb, a coordination, a quotation, a relative clause. Read the Tengwar first, then check yourself against the romanization beneath it.

```
mia thia whaline.
1SG 2SG thank.
(I thank you.)
```

```
pi su thia towe nai.
POL OPT 2SG well be.
(I hope that you are well.)
```

```
pi wa mia ha lo melu so po nila.
POL Q 1SG PROX PL friend FUT POT see.
(Might I see these friends?)
```

```
mia lopia ka nulae.
1SG child CAUS sleep.
(I make the child sleep.)
```

```
lopia se ka nulae.
child PASS CAUS sleep.
(The child is made to sleep.)
```

```
pheralu so ho shua.
rain FUT ASSUM come.
(Rain will probably come.)
```

```
pheralu to hi lepa.
rain PST DIR fall.
(Rain fell.)
```

```
shia to si ke thalo.
3SG PST IPFV INFER walk.
(They were walking.)
```

```
suliwa mua phitura nai.
snake LOC well be.
(A snake is at the well.)
```

```
mia lohau hi hea.
1SG dog DIR hear.
(I heard the dog.)
```

```
mia ruela mua muila nila.
1SG path LOC earth see.
(I see the trail in the earth.)
```

```
melu to ke wepu.
friend PST INFER go.
(The friend left.)
```

```
mia ko lothea sano.
1SG FOC love know.
(It is love that I know.)
```

```
wi himo piru mua silawo nai.
two HUM.CLF trader LOC village be.
(Two traders are at the village.)
```

```
lo mia mua womu wi shao ta himo miona nai.
PL 1SG LOC home two three-group one HUM.CLF person be.
(At home we are seven people.)
```

```
henoi sulopa nai.
ENOUGH soup be.
(There is enough soup.)
```

```
wia himo miona so shua.
how many HUM.CLF person FUT come.
(How many people will come?)
```

```
shiro phelora nela whalo nai.
tree beautiful COORD large be.
(The tree is beautiful and large.)
```

```
shia shola mia wepu sholo to haolu.
3SG QUOT.COMP 1SG go QUOT.COMP.CLOSE PST speak.
(They said, "I am leaving.")
```

```
mia mena shia to wepu meno sano.
1SG DECL.COMP 3SG PST go DECL.COMP.CLOSE know.
(I know that they left.)
```

```
rena mia to theo shelu phelora nai.
REL 1SG PST read book beautiful be.
(The book that I read is beautiful.)
```

```
rena shia to kealo nophi phelora nai.
REL 3SG PST create story beautiful be.
(The story that they created is beautiful.)
```
