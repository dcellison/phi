# Part 4: One nominal atom

The opener, payload, and closer occupy one noun-phrase position. Nothing inside supplies Phi tense, aspect, case, agreement, or word class.

## Proper name

```
mia ne patha Toronto patho nila.
1SG NAME EXT.EXACT [Toronto] EXT.EXACT.CLOSE see.
(I see Toronto.)
```

`ne` precedes `patha`, not the payload. It announces that the entire atom is a proper name.

## Preposition object

```
mia wei patha ResearchGate patho phelui sepho.
1SG DAT EXT.EXACT [ResearchGate] EXT.EXACT.CLOSE word send.
(I send a message to ResearchGate.)
```

The dative `wei` scopes over the complete frame. The ordinary adjunct-before-object order remains unchanged.

## Modifier or possessor

```
mia patha IPCC patho shelu theo.
1SG EXT.EXACT [IPCC] EXT.EXACT.CLOSE book read.
(I read the IPCC book.)
```

As with any Phi noun before another noun, context decides whether the relation is possession, source, type, or another nominal association.

## Subject and predicate

```
patha photosynthesis patho noetha nai.
EXT.EXACT [photosynthesis] EXT.EXACT.CLOSE essential be.
(Photosynthesis is essential.)

ha siru patha DOI:10.1000/182 patho nai.
PROX marker EXT.EXACT [DOI:10.1000/182] EXT.EXACT.CLOSE be.
(This marker is DOI:10.1000/182.)
```

The first frame is the subject. The second is a predicate before `nai`. An external verb would work no differently: keep it inside the atom and choose a Phi verb for the clause that discusses it.
