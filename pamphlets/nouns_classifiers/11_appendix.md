# Appendix: quick reference

## The canonical order

`[Possessor] [Quantifier] [Deictic] [Number or lo] [Classifier] [Descriptors] [Noun]`

The quantity positions are alternatives; a classifier appears only after a numeral or `wia`; no phrase fills every position.

## The claims ladder

| Expression | Claim |
|---|---|
| `melu` | number unstated; context carries |
| `lo melu` | more than one; no count |
| `ta melu` | exactly one |
| `wi melu` | exactly two |
| `ta shao melu` | exactly three |
| `shao melu` | approximately a three-group |
| `soli melu` | some; indefinite and nonzero |
| `sheloi melu` | contextually many |

`mu` counts an exact zero; `mawha` quantifies none.

## The quantifiers

| Phi | Gloss | Range |
|---|---|---|
| `theula` | UNIV | the whole relevant set |
| `sheloi` | MANY | contextually large |
| `shelami` | MOST | the larger part |
| `soli` | SOME | indefinite, nonzero |
| `phina` | FEW | contextually small |
| `theli` | EACH | one by one |
| `wheli` | ANY | free choice within the set |
| `mawha` | NONE | no member or amount |
| `henoi` | ENOUGH | sufficient for the purpose |
| `wia` | how many | the question |

## The classifiers

| Classifier | Gloss | Scope |
|---|---|---|
| `himo` | HUM.CLF | people |
| `lipha` | LIFE.CLF | other living beings |
| `themo` | THING.CLF | detached and crafted objects |
| `nophe` | ABST.CLF | concepts, time, events |

Always optional; placed after the complete number expression; the counting pamphlet drills the choosing.

## Working rules

- A bare noun is a complete reference; context carries identity and, when unmarked, number.
- One phrase takes one quantity marker: `lo`, a numeral, or a quantifier, never two together.
- `ta shao` counts an exact three; bare `shao` claims a scale. Say what you counted, estimate what you gauged.
- Classifiers ride on counts and on `wia`, never on ordinary quantifiers.
- Descriptors stack in the order that groups them; every adjective is also its quality noun.
- A noun before a noun modifies by association: owner, source, direction, kind, or character, resolved by context.
- The possessor leads its phrase; the having claim takes `phelu` with the possessed thing as object.
