# The pamphlets

The manual explains Phi's machinery once. These pamphlets give the difficult parts more room, with extended examples, repeated practice, exercises, and answers. Read one from beginning to end when a construction is new; later, its numbered files make a useful working reference.

The machine-readable [catalogue](catalogue.json) supplies the website shelf order and summaries. It also marks the Tengwar pamphlet for dual-script rendering. The site build stops if a directory or catalogue entry goes missing.
