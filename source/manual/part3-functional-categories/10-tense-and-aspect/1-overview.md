# Section 10: Tense and Aspect

## Overview

The **tense and aspect system** forms the temporal backbone of phi's verbal expression, providing speakers with precise tools for **temporal reference and aspectual distinctions**. This system combines a simple three-way tense distinction with a rich eight-way aspectual system, enabling nuanced temporal expression across narrative, academic, and conversational contexts.

## Core Function

**Temporal particles** answer fundamental questions about when and how actions occur:

- **When does it happen?** → Tense particles (`li`, `ta`, `su`)
- **How does it unfold?** → Aspect particles (`we`, `la`, `ni`, `po`, `pu`, `ri`, `wi`, `wu`)
- **What is the speaker's attitude?** → Modal particles (`to`, `ru`, `we`)
- **What is the polarity?** → Negation particle (`me`)

## The Temporal Architecture

Phi's temporal system operates through **Slot 1: Verb Phrase Particles** with systematic internal ordering:

**Tense > Aspect > Mood > Negation**

```
[Temporal] [Aspectual] [Modal] [Negative] [Verb]
mia li    la       ru      me     whera
1SG PST   PROG     OBLG    NEG    learn
"I was not supposed to be learning."
```

This ordering reflects **scope relationships** where broader temporal context precedes more specific aspectual and modal information.

## Cross-Linguistic Perspective

### Universal Temporal Categories

Most human languages distinguish:
- **Tense**: Past, present, future temporal reference
- **Aspect**: Perfective vs. imperfective action viewing
- **Modality**: Speaker attitudes toward propositions

### Phi's Innovations

**Clear syntactic separation**: Unlike fusional languages, phi separates tense from aspect
```
English: "was learning" (fused past + progressive)
Phi:     li la whera   (separate past + progressive particles)
```

**Rich aspectual system**: Eight aspectual distinctions exceed most natural languages
```
Standard aspectual systems: 2-4 aspects
Phi aspectual system:       8 aspects
```

**Systematic temporal-evidential interactions**: Evidentials license different temporal references
```
✅ hi mia ta whera.    (direct evidence + present)
✅ mu mia li whera.    (memory evidence + past)  
❌ hi mia su whera.    (direct evidence + future - problematic)
```

## The Complete Temporal System

### Tense Particles (3)

| Particle | Function | Temporal Reference | Example |
|----------|----------|-------------------|---------|
| `li` | Past | Before speech time | `mia li whera` "I learned" |
| `ta` | Present | At speech time | `mia ta whera` "I learn" |
| `su` | Future | After speech time | `mia su whera` "I will learn" |

### Aspect Particles (8)

| Particle | Function | Aspectual Meaning | Example |
|----------|----------|-------------------|---------|
| `we` | Desiderative | Want/desire to do | `mia we whera` "I want to learn" |
| `la` | Progressive | Ongoing action | `mia la whera` "I am learning" |
| `ni` | Perfect | Completed with relevance | `mia ni whera` "I have learned" |
| `po` | Habitual | Regular/repeated action | `mia po whera` "I usually learn" |
| `pu` | Perfective | Completed action | `mia pu whera` "I learned (completely)" |
| `ri` | Imperfective | Ongoing/incomplete | `mia ri whera` "I was learning" |
| `wi` | Inceptive | Beginning action | `mia wi whera` "I start learning" |
| `wu` | Cessative | Ending action | `mia wu whera` "I finish learning" |

## Functional Specializations

### Narrative Temporal Marking

**Sequential aspectual marking**: Stories use enhanced temporal-aspectual combinations
```
Narrative sequence:
mia li wi whera.  mia li la whera.  mia li wu whera.
1SG PST INCEP learn. 1SG PST PROG learn. 1SG PST CESS learn
"I started learning. I was learning. I finished learning."
```

**Habitual past patterns**: Background information in narratives
```
mia li po na he nowhea whelu.
1SG PST HAB OBJ HUM person love
"I used to love the person." (narrative background)
```

### Academic Temporal Precision

**Perfect aspect for research findings**:
```
ne moruphea ni riphe ta phera.
INAN findings PRF important PRS be
"The findings have proven important."
```

**Future predictive patterns**:
```
ro ne reshathi su mo ma naphe ta phera.
INF INAN research FUT CMPR EMPH precise PRS be
"Research will be *more* precise." (I infer this)
```

### Conversational Temporal Efficiency

**Reduced marking in casual speech**:
```
Formal:   mu mia li la te whera.
          MEM 1SG PST PROG VRB learn
          "I was learning." (I remember)

Casual:   mia li la whera.
          1SG PST PROG learn
          "I was learning."
```

## Systematic Interactions

### Tense-Aspect Combinations

**Productive combinations** create nuanced temporal meanings:

**Past + Aspects**:
```
mia li ni whera.     (past + perfect = pluperfect)
"I had learned."

mia li la whera.     (past + progressive = past continuous)
"I was learning."

mia li po whera.     (past + habitual = used to)
"I used to learn."
```

**Future + Aspects**:
```
mia su wi whera.     (future + inceptive = will start)
"I will start learning."

mia su po whera.     (future + habitual = will usually)
"I will usually learn."
```

### Evidential-Temporal Dependencies

**Compatible combinations**:
```
✅ mu mia li whera.   (memory + past)
✅ hi mia ta whera.   (direct + present)
✅ ro mia su whera.   (inference + future)
```

**Problematic combinations**:
```
❌ hi mia su whera.   (direct + future)
❌ mu mia ta whera.   (memory + present - marked)
```

### Modal-Temporal Integration

**Obligation across time**:
```
mia li ru whera.     (past obligation)
"I should have learned."

mia ta ru whera.     (present obligation)
"I should learn."

mia su ru whera.     (future obligation)
"I will need to learn."
```

## Processing and Acquisition

### Cognitive Advantages

**Incremental temporal interpretation**:
1. **Tense** establishes temporal frame
2. **Aspect** specifies temporal structure
3. **Mood** adds speaker attitude
4. **Negation** reverses polarity

**Predictable ordering** enables efficient parsing and production.

### Learning Progression

**Stage 1**: Master basic tense distinctions (`li`, `ta`, `su`)
**Stage 2**: Learn core aspects (`la`, `ni`, `po`)  
**Stage 3**: Integrate modal and negative combinations
**Stage 4**: Handle complex multi-aspect constructions
