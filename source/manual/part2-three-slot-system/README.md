## The Three-Slot Architecture

### Structural Framework

Phi organizes all particles into **three systematic slots** that work together to construct meaningful utterances:

```
[Slot 0: Sentence Frame] [Subject] [Slot 1: Verb Phrase] [Slot 2: Core Word] [Core Word]
```

**Example of full three-slot coordination**:
```
so hi wa    mia    ta la ru me    na he lo ma    nowhea    te whera
[Slot 0]    SUBJ   [Slot 1]       [Slot 2]       WORD      VRB WORD

"May I politely ask with direct evidence: was I not supposed to be 
progressively learning from *the* people?"

Slot 0: so (politeness) + hi (direct evidential) + wa (interrogative)
Slot 1: ta (present) + la (progressive) + ru (obligation) + me (negation)  
Slot 2: na (object) + he (human) + lo (plural) + ma (emphasis)
```

### Architectural Principles

**Systematic semantic scope**: Each slot has predictable scope relationships
**Compositional transparency**: Meaning builds systematically from left to right
**Functional specialization**: Each slot handles specific aspects of meaning construction
**Cross-slot integration**: Slots work together to create complex but transparent meanings

## Slot Functions and Organization

### Slot 0: Sentence Frame Particles
**Establishing discourse context and speaker stance**

| Category | Function | Examples | Position |
|----------|----------|----------|-----------|
| **Politeness** | Social register | `so` (polite) | Initial |
| **Evidentiality** | Knowledge source | `hi` (direct), `mu` (memory), `ro` (inference), `ke` (knowledge), `ra` (report), `wa` (assumption) | Second |
| **Interrogative** | Question marking | `wa` (general Q) | Final |

**Slot 0 ordering**: `[Politeness] [Evidential] [Interrogative]`

```
so hi wa mia ta whera.
POL DIR Q 1SG PRS learn
"May I politely ask with direct evidence: do I learn?"
```

### Slot 1: Verb Phrase Particles  
**Building temporal-modal-aspectual structure**

| Category | Function | Examples | Position |
|----------|----------|----------|-----------|
| **Tense** | Temporal reference | `li` (past), `ta` (present), `su` (future) | First |
| **Aspect** | Temporal structure | `la` (progressive), `ni` (perfect), `po` (habitual), `pu` (perfective), `ri` (imperfective), `wi` (inceptive), `wu` (cessative), `we` (desiderative) | Second |
| **Mood** | Speaker attitude | `ru` (obligation), `to` (imperative) | Third |
| **Negation** | Polarity | `me` (negative) | Final |

**Slot 1 ordering**: `[Tense] [Aspect] [Mood] [Negation]`

```
mia li la ru me whera.
1SG PST PROG OBLG NEG learn
"I was not supposed to be learning."
```

### Slot 2: Core Word Particles
**Specifying argument structure and emphasis**

| Category | Function | Examples | Position |
|----------|----------|----------|-----------|
| **Case** | Argument roles | `si` (subject), `na` (object), `te` (verb marker) | First |
| **Animacy** | Semantic categories | `he` (human), `pi` (animal), `ne` (inanimate) | Second |
| **Number** | Quantity | `wo` (paucal), `lo` (plural), `no` (greater plural) | Third |
| **Comparison** | Degree | `pa` (most), `mo` (more), `sa` (equal), `le` (less), `re` (least) | Fourth |
| **Emphasis** | Focus | `ma` (emphasis) | Final |

**Slot 2 ordering**: `[Case] [Animacy] [Number] [Comparison] [Emphasis]`

```
na he lo mo ma nowhea whelu.
OBJ HUM PL CMPR EMPH person love
"love *more* people"
```

## Systematic Integration Patterns

### Cross-Slot Semantic Coordination

**Evidential-temporal coherence** (Slot 0 + Slot 1):
```
mu mia li whera.        (memory + past = natural)
MEM 1SG PST learn
"I learned." (I remember past learning)

hi mia ta whera.        (direct + present = natural)  
DIR 1SG PRS learn
"I learn." (I directly observe current learning)

❌ hi mia su whera.     (direct + future = problematic)
   DIR 1SG FUT learn
   (Cannot directly observe future events)
```

**Politeness-complexity coordination** (Slot 0 + overall structure):
```
Formal contexts: so + complex particle combinations
Casual contexts: minimal particle combinations + flexible marking
```

**Emphasis-scope integration** (Slot 2 + semantic content):
```
ma + [focused element] = strategic emphasis placement
Modal particles + ma = emphasized modal relationships
Comparison particles + ma = emphasized degree relationships
```

## Structural Advantages

### Compositional Transparency

**Predictable meaning construction**: Each slot contributes systematic semantic components
```
Basic: mia whera.                    "I learn."
+Slot 0: hi mia whera.              "I learn." (direct evidence)
+Slot 1: hi mia ta la whera.        "I am learning." (+present +progressive)
+Slot 2: hi mia ta la na he ma nowhea whera. "I am learning from *the* person." 
         (+object +human +emphasis)
```

**Systematic scope relationships**: Predictable semantic scope across slots
```
Evidential scope > Temporal scope > Modal scope > Case scope > Focus scope
```

### Cognitive Processing Benefits

**Left-to-right incremental interpretation**:
1. **Slot 0**: Establish discourse frame and evidential stance
2. **Subject**: Identify primary participant  
3. **Slot 1**: Build temporal-modal structure
4. **Slot 2 + Core Word**: Integrate argument structure with semantic content

**Predictable complexity**: Systematic expansion from simple to complex constructions
**Regular patterns**: Same organizational principles across all semantic domains

### Pedagogical Advantages

**Systematic learning progression**:
1. **Single-slot mastery**: Learn one slot at a time
2. **Cross-slot coordination**: Practice systematic slot integration  
3. **Complex construction**: Build sophisticated meanings systematically
4. **Stylistic variation**: Use systematic principles for register and style effects

**Pattern recognition**: Same structural principles apply across all functional domains
**Error prediction**: Systematic organization enables systematic error correction

## Advanced Structural Patterns

### Multi-Clause Coordination

**Cross-clausal particle relationships**:
```
Clause 1: [Slot 0] [Subject] [Slot 1] [Slot 2] [Words]
Clause 2: [Slot 0] [Subject] [Slot 1] [Slot 2] [Words]

Example:
so hi mia ta whera mi ro sha li me whera.
POL DIR 1SG PRS learn CNTR INF 3SG PST NEG learn
"I politely learn (I directly observe), but s/he didn't learn (I infer)."
```

### Embedded Structure Patterns

**Systematic nesting and coordination**:
```
Complex constructions maintain three-slot organization at multiple levels
Subordinate clauses follow same structural principles
Relative constructions integrate systematically with main clause structure
```

### Register and Style Effects

**Systematic register variation through particle density**:
```
Academic formal: High particle density across all three slots
Conversational casual: Reduced particle density + context dependency
Literary artistic: Strategic particle density for stylistic effect
```

## Practical Applications

### Production Strategy

**Systematic utterance construction**:
1. **Determine discourse context** → Select Slot 0 particles
2. **Establish temporal-modal frame** → Select Slot 1 particles
3. **Specify argument structure** → Select Slot 2 particles  
4. **Integrate semantic content** → Coordinate particles with core words

### Comprehension Strategy

**Systematic interpretation process**:
1. **Process Slot 0** → Establish discourse frame and speaker stance
2. **Identify subject** → Determine primary participant
3. **Process Slot 1** → Build temporal-modal expectations
4. **Process Slot 2 + Words** → Integrate argument structure with semantic content

### Error Analysis and Correction

**Systematic error patterns**:
- **Slot confusion**: Particles in wrong slots
- **Ordering violations**: Incorrect within-slot ordering
- **Scope misunderstanding**: Misinterpreted particle scope relationships
- **Integration failures**: Poor cross-slot semantic coordination
