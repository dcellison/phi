# Section 5: Slot 0 - Sentence Frame Particles

## Overview

Slot 0 particles occupy the highest position in phi's particle hierarchy, appearing at the very beginning of clauses to establish the fundamental communicative context. These particles set the **sentence frame** - determining the overall type, evidential stance, politeness level, and discourse relationship of the entire utterance.

## Core Function

**Sentence framing** particles answer the essential question: *What kind of communicative act is this?*

- **Sentence type**: Is this a statement, question, or exclamation?
- **Evidential stance**: What is the source and reliability of this information?
- **Social register**: What level of politeness or formality is appropriate?
- **Discourse context**: How does this utterance relate to the ongoing conversation?

## The 13 Slot 0 Particles

### Sentence Type Markers (4 particles)
| Particle | Function | Example Context |
|----------|----------|-----------------|
| `wa` | Interrogative | Questions and information requests |
| `ho` | Exclamation | Emotional expressions and emphasis |
| `tu` | Conditional | Hypothetical situations and conditions |
| `hu` | Possibility | Uncertainty and speculation |

### Evidentiality System (6 particles)
| Particle | Function | Information Source |
|----------|----------|-------------------|
| `hi` | Direct Evidence | Personal observation/experience |
| `ro` | Inference | Logical deduction from evidence |
| `nu` | Hearsay | General second-hand information |
| `ti` | Reported Speech | Attributed to specific source |
| `mu` | Memory | Personal recollection |
| `pe` | Presumption | General knowledge/assumptions |

### Discourse Management (3 particles)
| Particle | Function | Discourse Role |
|----------|----------|----------------|
| `so` | Politeness | Social register and respect |
| `ha` | Topic | Thematic organization |
| `mi` | Contrast | Opposition and comparison |

## Structural Properties

### Position Rules
1. **Always sentence-initial**: Slot 0 particles must appear before all other elements
2. **Maximum one per clause**: Only one sentence frame particle per clause is permitted
3. **Scope over entire clause**: The particle's meaning applies to the complete grammatical unit

### Interaction Patterns
- **Evidentials + Tense**: Evidentials interact with temporal particles to create sophisticated epistemic-temporal meanings
- **Politeness + Question**: `so wa` creates polite interrogatives
- **Topic + Contrast**: `ha ... mi` patterns manage complex discourse relationships

## Cross-Linguistic Perspective

Phi's Slot 0 system synthesizes patterns found across human languages:

| Function | Natural Language Examples | Phi Implementation |
|----------|--------------------------|-------------------|
| Questions | Japanese `ka`, Mandarin `ma` | `wa` |
| Evidentiality | Turkish evidentials, Quechua system | Six-way distinction |
| Politeness | Korean honorifics, Thai particles | `so` |
| Topic marking | Japanese `wa`, Korean `-eun` | `ha` |

## Learning Strategy

**Recommended approach**:
1. **Start with evidentials**: These are obligatory in formal contexts
2. **Add sentence types**: Master questions and exclamations
3. **Incorporate discourse markers**: Learn topic and contrast patterns
4. **Refine with politeness**: Adjust for social appropriateness
