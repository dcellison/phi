# Modality System

## Modal Attitudes in Phi

Phi's modality system expresses **speaker attitudes** toward actions and states, covering commands, obligations, and desires. Modal particles in Slot 1 modify entire verb phrases to indicate whether actions are commanded, required, or wanted.

## The Complete Modal Paradigm

| Particle | Modality | English Gloss | Speaker Attitude | Usage Context |
|----------|----------|---------------|------------------|---------------|
| `to` | Imperative | "do X!" | Direct command | Orders, instructions |
| `ru` | Obligative | "should/must" | Moral/practical obligation | Duties, necessities |
| `we` | Desiderative | "want to" | Desire/intention | Wishes, plans |

*Note: `we` was covered comprehensively in the Aspect System (6.3) due to its dual aspectual-modal nature. This section focuses on purely modal particles.*

## Core Modal Categories

### 1. `to` - Imperative Modality

**Function**: Expresses direct commands, instructions, and requests for action.

**Core usage principles**:
- Direct orders and commands
- Polite requests (with `so`)
- Instructions and directions
- Urgent suggestions

**Basic imperative patterns**:
```
to whera.
IMP learn
"Learn!" / "Do learn!"

to na ne whethea whera.
IMP OBJ INAN book learn
"Learn from books!"

to shio phiato whera.
IMP from this learn
"Learn from this!"
```

**Polite imperatives** (with `so`):
```
so to whera.
POL IMP learn
"Please learn."

so to na ne lophui whona.
POL IMP OBJ INAN art look
"Please look at art."

so to monite sharo.
POL IMP there go
"Please go there."
```

**Complex imperatives**:
```
to li whera mi to su whema.
IMP PST learn CNTR IMP FUT work
"Learn (from the past) but work (in the future)!"

to ma riphe whera.
IMP EMPH important learn
"Learn *importantly*!"
```

### 2. `ru` - Obligative Modality

**Function**: Expresses obligation, necessity, and moral/practical duties.

**Core usage principles**:
- Moral obligations
- Practical necessities
- Social expectations
- Logical requirements

**Basic obligative patterns**:
```
mia ru whera.
1SG OBLG learn
"I should learn." / "I must learn."

sha ru na ne whethea whera.
3SG OBLG OBJ INAN book learn
"S/he should learn from books."

lo mia ru na thueta whisi.
PL 1SG OBLG OBJ that see
"We should see that."
```

**Temporal obligations**:
```
mia li ru whera.
1SG PST OBLG learn
"I should have learned."

mia su ru whera.
1SG FUT OBLG learn
"I will need to learn."

mia ta ru whera.
1SG PRS OBLG learn
"I should learn (now)."
```

**Degrees of obligation**:
```
mia ru whera.           (standard obligation)
1SG OBLG learn
"I should learn."

ma ru mia whera.        (emphatic obligation)
EMPH OBLG 1SG learn
"I *must* learn."

so ru mia whera.        (polite obligation)
POL OBLG 1SG learn
"I should politely learn."
```

## Modal Combinations and Interactions

### Modality + Tense
Modal particles combine systematically with tense:

#### Past Obligations
```
mia li ru whera.        (PST + OBLG)
1SG PST OBLG learn
"I should have learned."

sha li ru na ne lophui whona.
3SG PST OBLG OBJ INAN art look
"S/he should have looked at art."
```

#### Future Commands
```
to su whera.            (IMP + FUT)
IMP FUT learn
"Go learn!" (future-oriented command)

to su na ne whethea whera.
IMP FUT OBJ INAN book learn
"Go learn from books!"
```

### Modality + Aspect
Modal and aspectual particles can co-occur:

```
mia ru la whera.        (OBLG + PROG)
1SG OBLG PROG learn
"I should be learning."

to pu whera.            (IMP + PFV)
IMP PFV learn
"Learn completely!"

mia ru ni whera.        (OBLG + PRF)
1SG OBLG PRF learn
"I should have learned (by now)."
```

### Modality + Negation
Negation interacts with modality to create prohibition and non-obligation:

```
to me whera.            (prohibition)
IMP NEG learn
"Don't learn!"

mia ru me whera.        (negative obligation)
1SG OBLG NEG learn
"I should not learn."

me ru mia whera.        (non-obligation)
NEG OBLG 1SG learn
"I don't have to learn."
```

## Social and Pragmatic Functions

### Commands and Politeness
Imperatives range from direct to highly polite:

**Direct command**:
```
to whera.
IMP learn
"Learn!"
```

**Polite request**:
```
so to whera.
POL IMP learn
"Please learn."
```

**Formal instruction**:
```
so to na ne whethea ta whera.
POL IMP OBJ INAN book PRS learn
"Please learn from books."
```

### Obligations and Social Expectations

**Personal duty**:
```
mia ru whera.
1SG OBLG learn
"I should learn."
```

**Social expectation**:
```
he nowhea ru whera.
HUM person OBLG learn
"A person should learn."
```

**Universal obligation**:
```
lo he nowhea ru whera.
PL HUM person OBLG learn
"People should learn."
```

## Register Variation

### Formal Context
Explicit modal marking with politeness:
```
so ro he nowhea ru na ne whethea ta whera.
POL INF HUM person OBLG OBJ INAN book PRS learn
"I infer that people should politely learn from books."
```

### Informal Context
Simplified modal expressions:
```
to whera.               (direct)
IMP learn
"Learn!"

ru whera.               (implied subject)
OBLG learn
"Should learn."
```

### Educational Context
Clear instructional modality:
```
to phiato whera. to thueta whemo. to riphe phera.
IMP this learn. IMP that think. IMP important be.
"Learn this. Think about that. Be important."
```

## Cross-Linguistic Perspectives

### Modal Universals
Phi's modal system reflects universal patterns:
- **Imperative**: Found in all human languages
- **Obligative**: Common in languages with rich modal systems
- **Desiderative**: Widespread in particle-based languages

### Typological Comparisons
- **English**: Modal auxiliaries (should, must, want)
- **Japanese**: Modal endings and particles
- **German**: Modal particles and verbs
- **Mandarin**: Modal particles (should 应该, want 要)

## Advanced Modal Usage

### Modal Stacking
Multiple modal meanings in sequence:
```
so ru we whera.         (obligation + desire)
POL OBLG DES learn
"One should politely want to learn."
```

### Modal Discourse
Modal particles in extended discourse:
```
to whera. ru whera. we whera.
IMP learn. OBLG learn. DES learn.
"Learn! (You) should learn. (You should) want to learn."
```

### Conditional Modality
Modality in hypothetical contexts:
```
tu mia ru whera, mia su riphe phera.
COND 1SG OBLG learn, 1SG FUT important be
"If I should learn, I will be important."
```

## Pedagogical Recommendations

### Learning Sequence
1. **Basic commands**: `to` for simple instructions
2. **Polite requests**: `so to` for social appropriateness  
3. **Obligations**: `ru` for expressing duties
4. **Complex combinations**: Multiple particles together

### Common Errors
- **Position**: Modals must precede verbs in Slot 1
- **Scope**: Modals apply to entire verb phrase
- **Politeness**: Use `so` with commands for appropriate register
