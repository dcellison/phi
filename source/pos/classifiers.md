# classifiers

> classifiers are words that categorize nouns based on their physical characteristics, shape, function, or other semantic features. they are used in many languages to indicate quantity, grouping, or measurement of objects, and often appear between numerals and the nouns they classify.

## introduction to phi classifiers

classifiers in phi adhere to the structure `[C][P]` where:

- C = consonant (h, l, m, n, p, r, s, t, w)
- P = vowel pair (ia, io, iu, ie, etc., excluding identical pairs like ii)

this pattern creates words like `lea` (long objects), `moi` (flat objects), and `teo` (round objects), which always begin with a consonant followed by a vowel pair. this gives classifiers a concise, expressive sound that is perfect for their frequent use in counting and categorization contexts.

classifiers in phi are used immediately after numbers and quantifiers, but before the noun they classify. in phi's SOV word order, the structure follows: number + classifier + noun. examples: `phi lea liphai` (one [long] tree), `whu rao whithoa` (two [animal] dogs), `pho teo phuawhai` (four [round] stones).

while classifiers share the same phonotactic pattern as interjections, phi maintains a strict lexical distinction where no classifier ever overlaps with an interjection. this upholds phi's single part of speech principle while providing systematic categorization capabilities.

phi's classifier system is designed to be intuitive and minimal, focusing on the most fundamental categorization concepts found across human languages. this reduces learning burden while maintaining expressive power for essential classification needs.

## design philosophy

the phi classifier system prioritizes **essential categorization** over exhaustive specification. this design:

- focuses on universal classification concepts found across classifier languages
- eliminates redundancy by providing one clear classifier per semantic domain
- covers fundamental categorization needs for daily communication
- maintains systematic phonotactic integrity
- enables precise counting and grouping without overwhelming complexity

research into classifier typology shows that certain categorization concepts are universal across languages with classifier systems. phi's classifier system concentrates on these essential categories while avoiding overly specific or culturally-bound distinctions.

## usage examples

in phi, classifiers follow the pattern: number + classifier + noun

### basic counting
```
phi lea liphai
one CL.long tree
One tree
```

```
whu rao whithoa
two CL.animal dog
Two dogs
```

```
pho teo phuawhai
four CL.round stone
Four stones
```

### with different numbers
```
shu hau thephoa
seven CL.human person
Seven people
```

```
thuphi moi whethea
thousand CL.flat paper
One thousand sheets of paper
```

### contextual usage
the same noun may take different classifiers based on context:
- `phi lea liphai` (one [long] tree) - standing tree
- `phi moi liphai` (one [flat] tree) - fallen/cut tree viewed as flat object

## usage notes

**natural usage principles**:

phi embodies the **minimal marking principle** - use only what's needed for clear communication. this creates authentic, flowing speech patterns rather than rigid categorization structures.

**context sensitivity**: classifiers can be omitted when categorization is obvious from context or the noun itself. this includes dropping animacy when the noun clearly indicates life form, shape when physical form is evident, or function when purpose is apparent.

**progressive complexity**: beginners use basic counting; advanced speakers add precise categorization only when communicatively valuable.

**examples with natural simplification**:

**Full form:**
```
whu hau thephoa
two CL.human person
Two human people
```

**Natural form:**
```
whu thephoa
two person
Two people
```

**Full form:**
```
phi teo phuawhai
one CL.round stone
One round stone
```

**Natural form:**
```
phi phuawhai
one stone
One stone
```

**beginner-friendly usage**: 
- Start with simple counting: `phi noshea` (one food)
- Add classifiers gradually: 
```
phi sae noshea
one CL.food food
One food item
```
- Context often eliminates need for obvious categorization

**semantic precision**: each classifier covers one clear categorization domain without overlap. objects are classified by their most salient feature in context.

**SOV positioning**: classifiers appear between numbers and nouns in phi's subject-object-verb word order. examples: `whu tau thephoa` (two [group] people), `phi sou noshea` (one [mass] food).

**contextual flexibility**: the same object may take different classifiers depending on how it's perceived or used in context, allowing speakers to highlight different aspects of categorization.

**no derivational morphology**: phi classifiers are not derived from other parts of speech. each maintains its distinct phonotactic pattern and semantic independence.
