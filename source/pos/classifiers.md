# classifiers

> classifiers are words that categorize nouns based on their physical characteristics, shape, function, or other features. they are used in many languages to indicate the quantity, grouping, or measurement of objects.

## introduction to phi classifiers

all classifiers in phi adhere to the structure `[C][P]` where:

- C = consonant (h, l, m, n, p, r, s, t, w)
- P = vowel pair (ia, io, iu, ie, etc., excluding identical pairs like ii)

this pattern creates words like `lea` (classifier for long objects), `moi` (classifier for 
flat objects), and `teo` (classifier for round objects), which always begin with a consonant 
followed by a vowel pair. this gives classifiers a concise, expressive sound that is 
perfect for their frequent use in counting contexts.

while classifiers share the same phonotactic pattern as interjections, phi maintains 
a strict lexical distinction where no classifier ever overlaps with an interjection. 
classifiers in phi are used immediately after numbers and quantifiers, but before the noun 
they classify. for example, "phi neo siwhea" (one [house classifier] house) or 
"whu rao thiphea" (two [animal classifier] animals). this structure helps speakers 
organize and categorize objects in a systematic way.

phi's classifier system is designed to be intuitive and minimal, reducing the learning 
burden while maintaining expressive power. the distinct lexical items ensure 
that classifiers are never confused with interjections, upholding phi's single part 
of speech principle.

## examples of classifier usage

in phi, classifiers are placed between numbers and nouns:

| phi example          | htructure | english translation            |
| -------------------- | -------------------------- | ----------------------------- |
| phi lea liphai       | one + CL.long + tree       | one tree                      |
| whu rao thiphea      | two + CL.animal + animal   | two animals                   |
| pho teo mophui       | four + CL.round + ball     | four balls                    |
| shu sae noshea       | seven + CL.food + food     | seven food items              |
| wha heu whoshia      | five + CL.vehicle + car    | five cars                     |
| phi hau thaphea      | one + CL.human + woman     | one woman                     |
| toa wou methuwa      | ten + CL.electronic + phone| ten phones                    |
| whu toa piu taothea  | twenty + CL.bundle + flower| twenty bundles of flowers     |

some nouns typically go with specific classifiers, while others might take 
different classifiers based on context. for example, "phi lea liphai" (one tree)
uses `lea` for a standing tree as a long object, but "phi pio whethea" (one book)
uses `pio` for the rectangular shape of the book.

## remaining classifier count

phi classifiers follow the pattern `[C][P]` where:
- C = consonant (h, l, m, n, p, r, s, t, w) - 9 possibilities
- P = vowel pair (ia, io, iu, ie, etc.) - 20 possibilities

Total possible combinations = 9 × 20 = 180 combinations

Currently defined classifiers = 37 (9 physical + 6 animate + 8 functional + 
6 spatial/temporal + 6 abstract + 6 quantitative)

Remaining available interjection and classifier words = 90

note: this pattern is shared between classifiers and interjections, but phi maintains
a strict lexical distinction where no word serves both functions, upholding the 
single part of speech principle that is core to phi's design.
