# lowhai riwhea phia la whuru
## Phi Translation Exercise: "Blowin' in the Wind"

> This is a Phi language translation exercise demonstrating complex sentence 
> structures, subordination, and tense usage. All sentences have been validated 
> through the Phi sentence validator for grammatical correctness.

### Translation Structure

Each pair of lines forms one complete sentence using subordination with `pimo` (before).

---

## Verse 1

**Line 1:**
```
hamite huphui thephoa ru shola
how-many road person OBLG walk
How many roads must a person walk
```

**Line 2:**
```
pimo thi thephoa su shuna
before 2sg person FUT call
Before you will call them a person
```

## Verse 2

**Line 3:**
```
hamite latheo whithea ru sharo
how-many sea dove OBLG sail
How many seas must a dove sail
```

**Line 4:**
```
pimo sha phia shuwhia su whuwe
before 3sg in sand FUT sleep
Before she will sleep in the sand
```

## Verse 3

**Line 5:**
```
hamite phothea ru whawi
how-many cannonball OBLG fly
How many cannonballs must fly
```

**Line 6:**
```
pimo sha walito su thunu
before 3sg forever FUT ban
Before they will be forever banned
```

## Chorus

**Line 7:**
```
lowhai mia luthea riwhea phia la whuru
answer my friend wind in PROG blow
The answer, my friend, is blowin' in the wind
```

**Line 8:**
```
lowhai riwhea phia la whuru
answer wind in PROG blow
The answer is blowin' in the wind
```

## Grammar Notes

### Key Vocabulary Used
- `lowhai` - answer (noun)
- `riwhea` - wind (noun) 
- `whuru` - blow (verb)
- `la` - progressive aspect particle
- `pimo` - before (subordinating conjunction)
- `ru` - obligative mood (must/should)
- `su` - future tense

### Subordinate Clause Structure
The translation uses `pimo` (before) + future tense (`su`) to express the hypothetical conditions in the original English questions.

### Progressive Aspect
The famous "blowin' in the wind" phrase uses the progressive aspect particle `la` to show the answer is actively blowing within the wind: `lowhai riwhea phia la whuru` (answer [is] blowing in [the] wind).

## Cultural Notes

This translation demonstrates Phi's systematic approach to:
- **Gender neutrality**: Using `thephoa` (person) rather than gendered terms
- **Temporal precision**: Clear distinction between obligative and future tenses
- **Subordination**: Explicit marking of logical relationships with conjunctions
- **Philosophical expression**: Maintaining poetic meaning through systematic grammar

The translation preserves the philosophical questioning nature of the original while 
showcasing Phi's capacity for complex, nuanced expression.
