# lesson 04: precision

> this lesson introduces the tools for precise specification in phi: determiners, 
> numbers, classifiers, and comparison. you'll learn to express exact quantities, 
> specificity, and degrees with systematic clarity.

## review: your comprehensive toolkit

from lessons 01-03, you now recognize these patterns:
- **particles** (shortest): `li`, `su`, `wa`, `me`, `ta`, `la`, `po`, `lu`, `nene`
- **verbs** (start with fricative): `phola`, `phamo`, `phose`, `phuwe`, `pheru`, `phenu`, `phemo`
- **adjectives** (fricative inside): `hashe`, `mipho`, `teshe`, `pisha`
- **nouns** (end fricative + vowel pair): `nuthui`, `thephoa`, `siwhea`, `wheisha`
- **adverbs** (3 syllables, no fricatives): `hamite`, `welime`, `timane`, `tapine`, `walime`
- **prepositions** (fricative + vowel pair): `thue`, `wheo`, `shio`, `phae`

## new pattern: determiners

determiners specify which things you mean and follow a **fricative + vowel pair + consonant + vowel** pattern:

### determiner pattern: fricative-vowel-pair-consonant-vowel

| determiner | breakdown | meaning |
|------------|-----------|---------|
| `phiato` | ph-ia-t-o | this |
| `thueta` | th-ue-t-a | that |
| `whieso` | wh-ie-s-o | which |
| `shoata` | sh-oa-t-a | all |

**recognition cue**: longer than prepositions, starts with fricative + vowel pair

## numbers and counting

phi has systematic number patterns:

### digits: fricative + vowel

| number | breakdown | meaning |
|--------|-----------|---------|
| `phi` | ph-i | one |
| `whu` | wh-u | two |
| `the` | th-e | three |
| `pho` | ph-o | four |
| `wha` | wh-a | five |

### magnitudes: fricative + vowel + fricative + vowel

| magnitude | breakdown | meaning |
|-----------|-----------|---------|
| `phitha` | ph-i-th-a | ten |
| `shupho` | sh-u-ph-o | hundred |
| `thuphi` | th-u-ph-i | thousand |

**recognition cue**: digits are short (fricative + vowel), magnitudes are longer

## classifiers for counting

classifiers group objects by shape/type and follow a **consonant + vowel pair** pattern:

### classifier pattern: consonant-vowel-pair

| classifier | breakdown | meaning |
|------------|-----------|---------|
| `teo` | t-eo | round objects |
| `lea` | l-ea | long objects |
| `moi` | m-oi | flat objects |
| `hau` | h-au | people/humans |

**recognition cue**: consonant + two vowels, shorter than adverbs

## using determiners

determiners come first in noun phrases to specify which things:

### basic specificity

```
phiato nuthui
this   pebble
this pebble

thueta siwhea
that   house
that house

whieso thephoa
which  person
which person

shoata wheisha
all    water
all water
```

### determiners with descriptions

```
phiato hashe nuthui
this   green pebble
this green pebble

thueta teshe mipho siwhea
that   good blue   house
that good blue house
```

## counting with numbers and classifiers

### basic counting structure
```
(number) (classifier) (noun)
```

```
phi teo nuthui
one round pebble
one pebble

whu hau thephoa
two people person
two people

the lea siwhea
three long house
three houses (long buildings)
```

### larger numbers

```
phi phitha teo nuthui
one ten    round pebble
ten pebbles

whu shupho hau thephoa
two hundred people person
two hundred people
```

## comparison and degree

use comparison particles to show relative qualities:

### comparison particles
- `mo` - comparative (more)
- `pa` - superlative (most)
- `sa` - equality (as/equally)

### making comparisons

```
phiato mo pisha nuthui
this   more soft pebble
this pebble is softer

thueta pa teshe siwhea
that   most good house
that house is best

mia sa hashe thephoa
I   equal green person
I am as green as the person
```

## complex noun phrases

combine all elements for precise specification:

### full noun phrase structure
```
(determiner) (number) (classifier) (adjective) noun
```

```
phiato whu teo hashe nuthui
this   two round green pebble
these two green pebbles

thueta phi hau teshe thephoa
that   one people good person
that one good person

shoata the lea pisha siwhea
all    three long soft house
all three soft long houses
```

## advanced sentence building

### precise descriptions with possession

```
mia phiato phi teo hashe nuthui thue mia wheo siwhea phose
I   this   one round green pebble of   I   at   house  see
I see this one green pebble of mine at the house

thueta whu hau thephoa li shio walime tapine phola
that   two people person PST from outside quickly walk
Those two people walked quickly from outside
```

### comparisons in context

```
phiato mo teshe siwhea thueta siwhea
this   more good house  that   house
This house is better than that house

lu mia phose thephoa pa hashe shoata thephoa
REL I  see   person  most green all    person
The person whom I see is the greenest of all people
```

## practice conversations

### conversation 1: shopping/selecting

**person a:**
```
wa thi whieso nuthui phamo
Q  you which  pebble like
Which pebble do you like?
```

**person b:**
```
mia phiato phi teo hashe nuthui phamo
I   this   one round green pebble like
I like this one green pebble
```

**person a:**
```
wa thueta nuthui hamite
Q  that   pebble how
How about that pebble?
```

**person b:**
```
thueta nuthui mo pisha
that   pebble more soft
That pebble is softer
```

### conversation 2: comparing

**person a:**
```
wa whieso pa teshe siwhea
Q  which  most good house
Which house is best?
```

**person b:**
```
phiato mipho siwhea pa teshe shoata siwhea
this   blue  house  most good all    house
This blue house is best of all houses
```

**person a:**
```
wa hamite thi phemo
Q  how    you think
What do you think?
```

**person b:**
```
mia sa phemo
I   equal think
I think the same
```

- `lea` - long objects

- `mo` - comparative (more)
- `pa` - superlative (most)
