# lesson 01: foundations

> this lesson introduces the essential patterns that unlock basic communication in 
> phi. by the end, you'll understand phi's core systematic principle and be able 
> to construct simple sentences.

## the key insight: sound reveals function

phi's most important feature is that **how a word sounds tells you what it does**. 
unlike english where "run" can be a noun or verb, every phi word has exactly one 
grammatical function that you can identify by its sound pattern.

### essential sound patterns

learn these four patterns first - they cover 80% of basic communication:

| pattern | function | examples | recognition cue |
|---------|----------|----------|-----------------|
| **consonant + vowel** | particle (grammar words) | `li` (past), `wa` (yes/no question) | shortest words |
| **starts with fricative** | verb (action words) | `phola` (walk), `phamo` (like) | begins with ph/wh/th/sh |
| **has fricative inside** | adjective (describing words) | `hashe` (green), `mipha` (beautiful) | fricative in middle |
| **ends with fricative + vowel pair** | noun (thing words) | `nuthui` (pebble), `thephoa` (person) | ends like -thui, -phoa |

*fricatives: ph, wh, th, sh (the "breathy" sounds)*

## essential vocabulary

### core particles (grammar tools)
- `li` - past tense ("did")
- `su` - future tense ("will") 
- `wa` - yes/no question marker
- `me` - negation ("not")

### pronouns
- `mia` - I/me
- `thi` - you

### starter nouns (things)
- `nuthui` - pebble/stone
- `thephoa` - person
- `siwhea` - house
- `wheisha` - water

### starter verbs (actions)
- `phola` - walk
- `phamo` - like
- `phose` - see
- `phuwe` - sleep

### starter adjectives (descriptions)
- `hashe` - green
- `mipho` - blue
- `teshe` - good
- `pisha` - soft

## basic sentence construction

### word order: subject-object-verb (SOV)

phi uses SOV order - the action comes last:

```
mia nuthui phamo
I   pebble like
I like the pebble
```

### time particles come before verbs

```
mia nuthui li phamo
I   pebble PST like
I liked the pebble

mia nuthui su phamo
I   pebble FUT like
I will like the pebble
```

### questions with wa (yes/no)

for yes/no questions, add `wa` at the beginning:

```
wa mia nuthui phamo
Q  I   pebble like
Do I like the pebble?

wa thi siwhea phose
Q  you house  see
Do you see the house?
```

*note: other question types (like "where" and "how") use different patterns - you'll learn these in lesson 02*

### negation with me

```
mia nuthui me phamo
I   pebble NEG like
I don't like the pebble

thi li me phola
you PST NEG walk
You didn't walk
```

## practice sentences

read these aloud, paying attention to the sound patterns:

### statements
```
mia phola
I   walk
I walk

thi thephoa phose
you person  see
You see the person

thephoa hashe siwhea phamo
person  green house  like
The person likes the green house
```

### past tense
```
mia li phola
I   PST walk
I walked

thi wheisha li phose
you water   PST see
You saw the water

thephoa li phuwe
person  PST sleep
The person slept
```

### questions
```
wa thi phola
Q  you walk
Do you walk?

wa mia teshe thephoa phose
Q  I   good  person  see
Do I see a good person?

wa thephoa li phuwe
Q  person  PST sleep
Did the person sleep?
```

### negative
```
mia me phola
I   NEG walk
I don't walk

thi siwhea me phose
you house  NEG see
You don't see the house

thephoa wheisha me phamo
person  water   NEG like
The person doesn't like water
```

## pattern recognition exercises

### identify the word types by sound:
1. `phamo` - ?
2. `nuthui` - ?
3. `hashe` - ?
4. `li` - ?

### build sentences using SOV order:
5. I + see + person → ?
6. you + past + walk → ?
7. question + person + like + water → ?

### answers:
1. verb (starts with fricative ph)
2. noun (ends with fricative + vowel pair)
3. adjective (fricative sh inside)
4. particle (consonant + vowel)
5. ```
   mia thephoa phose
   I   person  see
   I see the person
   ```
6. ```
   thi li phola
   you PST walk
   You walked
   ```
7. ```
   wa thephoa wheisha phamo
   Q  person  water   like
   Does the person like water?
   ```

## your first conversation

try this simple exchange:

**person a:** 
```
mimia
hello
Hello
```

**person b:** 
```
mimia
hello
Hello
```

**person a:** 
```
welime thi phola
where  you walk
Where do you walk?
```

**person b:** 
```
mia siwhea phola
I   house  walk
I walk to the house
```

**person a:** 
```
teshe
good
Good
```

**person b:** 
```
hamite thi phola
how    you walk
How do you walk?
```

*note: `welime` (where) and `hamite` (how) are adverbs - you'll learn their pattern in lesson 02*

## key takeaways

1. **sound = function**: word patterns immediately tell you grammatical role
2. **SOV order**: subject-object-verb, with particles before verbs
3. **particles do grammar**: `li`, `su` handle tense; `wa` handles yes/no questions; `me` handles negation
4. **pattern recognition**: once you know the patterns, new words are immediately categorizable
