# lesson 02: expanding

> this lesson expands your phi toolkit with adverbs, more particles, and systematic 
> modification patterns. you'll learn to build longer, more precise sentences while 
> maintaining phi's systematic clarity.

## review: the foundation

from lesson 01, you learned that sound reveals function:
- **particles** (shortest): `li`, `su`, `wa` (yes/no questions), `me`
- **verbs** (start with fricative): `phola`, `phamo`, `phose`
- **adjectives** (fricative inside): `hashe`, `mipho`, `teshe`
- **nouns** (end fricative + vowel pair): `nuthui`, `thephoa`, `siwhea`

## new pattern: adverbs

adverbs have a distinctive **three-syllable pattern with no fricatives**:

### adverb pattern: consonant-vowel-consonant-vowel-consonant-vowel

| adverb | breakdown | meaning |
|--------|-----------|---------|
| `hamite` | h-a-m-i-t-e | how |
| `welime` | w-e-l-i-m-e | where |
| `timane` | t-i-m-a-n-e | when |
| `tapine` | t-a-p-i-n-e | quickly |
| `walime` | w-a-l-i-m-e | outside |

**recognition cue**: longest common words (3 syllables), no fricatives anywhere

## adverbs in action

adverbs modify verbs and come **before** the verb they modify:

### asking questions with interrogative adverbs

unlike the yes/no questions you learned in lesson 01 (which use `wa`), these interrogative adverbs create questions by themselves:

```
hamite thi phola
how    you walk
How do you walk?

welime thi phola
where  you walk
Where do you walk?

timane thi phola
when   you walk
When do you walk?
```

*notice: no `wa` needed! the interrogative adverb alone makes it a question*

## phi's systematic question types

phi has two clean, systematic question patterns:

### yes/no questions (from lesson 01)
add `wa` to any statement:
```
wa thi phola (Do you walk?)
wa mia nuthui phamo (Do I like the pebble?)
```

### wh-questions (new in this lesson)
use interrogative adverbs alone:
```
hamite thi phola (How do you walk?)
welime thi phola (Where do you walk?)
timane thi phola (When do you walk?)
```

this systematic distinction keeps each question type clean and predictable!

### describing actions

```
mia tapine phola
I   quickly walk
I walk quickly

thephoa walime phose
person  outside see
The person sees outside
```

## more useful particles

expand your particle toolkit with these essential additions:

### time and aspect particles
- `ta` - present/ongoing
- `la` - progressive (currently doing)
- `po` - habitual (usually does)

### basic conjunction
- `nene` - and (connects things)

## particles in combination

particles follow a specific order: **tense > aspect > negation**

```
mia ta phola
I   PRES walk
I walk (now)

mia la phola
I   PROG walk
I am walking

mia li la phola
I   PST PROG walk
I was walking

mia po phola
I   HAB walk
I usually walk

mia po me phola
I   HAB NEG walk
I usually don't walk
```

## systematic adjective placement

adjectives go **before** the noun they describe, following this order:

### basic noun phrase structure
```
(adjective) noun
```

```
hashe nuthui
green pebble
green pebble

pisha siwhea
soft house
soft house

teshe thephoa
good person
good person
```

### multiple adjectives
when using multiple adjectives, they all come before the noun:

```
hashe pisha nuthui
green soft  pebble
green soft pebble

teshe mipho siwhea
good blue   house
good blue house
```

## building longer sentences

### connecting with nene (and)

```
mia phola nene thi phuwe
I   walk  and  you sleep
I walk and you sleep

thephoa hashe nuthui phamo nene mipho wheisha phose
person  green pebble like  and  blue  water   see
The person likes green pebbles and sees blue water
```

### combining adverbs with descriptions

```
mia tapine hashe siwhea phola
I   quickly green house  walk
I quickly walk to the green house

timane thephoa teshe wheisha phose
when   person  good  water   see
When does the person see good water?
```

### time combinations

```
thephoa li tapine phola
person  PST quickly walk
The person walked quickly

mia su walime teshe thephoa phose
I   FUT outside good person  see
I will see a good person outside
```

## pattern recognition exercises

### identify the word types:
1. `tapine` - ?

3. `la` - ?
4. `walime` - ?

### build sentences:
5. I + usually + quickly + walk → ?
6. question + person + when + green + water + see → ?
7. you + and + I + outside + walk → ?

### answers:
1. adverb (three syllables, no fricatives)
2. conjunction (two syllables, no fricatives)
3. particle (consonant + vowel)
4. adverb (three syllables, no fricatives)
5. ```
   mia po tapine phola
   I   HAB quickly walk
   I usually walk quickly
   ```
6. ```
   timane thephoa hashe wheisha phose
   when   person  green water   see
   When does the person see green water?
   ```
7. ```
   thi nene mia walime phola
   you and  I   outside walk
   You and I walk outside
   ```

## key insights from this lesson

1. **adverb pattern**: three syllables, no fricatives, modify verbs
2. **systematic questions**: `wa` for yes/no, interrogative adverbs alone for wh-questions
3. **particle order**: tense > aspect > negation
4. **adjective placement**: always before nouns
5. **systematic building**: each new element follows predictable patterns
