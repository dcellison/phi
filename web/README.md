# Phi lexicon explorer

A static, read-only view over the repository, which stays the single source of truth. Two pages: `index.html`, the landing — kia.md rendered at build time — and `explore.html`, the searchable lexicon. Both `index.html` and `lexicon.json` are generated by the build script and gitignored.

Local preview:

```bash
python3 scripts/build_explorer.py
python3 -m http.server -d web
```

then open http://localhost:8000. Deployed automatically to GitHub Pages on every push to main (`.github/workflows/pages.yml`).
