# Headed relative clauses

A headed relative clause modifies an explicit noun. The noun, called the "head," appears after the clause and receives the description. This is the most common type of relative clause, used whenever you want to specify which particular thing you're talking about.

## Basic examples

The subject describes the object:

```
rena mia lothea miona
REL 1SG love person
(the person whom i love)
```

The object describes the subject:

```
rena lopia shomela miona
REL child teach person
(the person who teaches children)
```

A state describes the noun:

```
rena serao nai shelu
REL old be book
(the book that is old)
```

A location describes the noun:

```
rena toremoa nia nai womu
REL mountain on be home
(the home that is on the mountain)
```

## Using relative clauses in sentences

Relative clauses function as noun phrases. They can serve as subject, object, or part of a prepositional phrase:

**As subject:**

```
rena mia to nila miona shia nai
REL 1SG PST see person 3SG be
(the person whom i saw is here.)
```

The entire phrase `rena mia to nila miona` serves as the subject of `nai`.

**As object:**

```
mia rena shia to kealo nophi theo
1SG REL 3SG PST create story read
(i read the story that they created.)
```

The relative clause `rena shia to kealo nophi` modifies "story," and the whole noun phrase is the object of `theo`.

**With prepositions:**

```
mia rena shelira mua nai ruela nia thalo
1SG REL forest LOC be path on walk
(i walk on the path that is in the forest.)
```

## Complex descriptions

Relative clauses can include their own modifiers, objects, and adverbials:

```
rena mia serao shelira mua to thalo ruela
REL 1SG old forest LOC PST walk path
(the path that i walked in the old forest)
```

```
rena shia lopia wei phelora nophi to haolu miona
REL 3SG child to beautiful story PST speak person
(the person who told a beautiful story to the child)
```

The key is that everything before the head noun is part of the description. No matter how complex the clause becomes, the structure remains the same: `rena` opens it, the clause unfolds, the noun closes it.

## Stacking descriptions

A noun can receive multiple descriptions by stacking adjectives and relative clauses:

```
rena mia to nila serao phelora shelu
REL 1SG PST see old beautiful book
(the old beautiful book that i saw)
```

All modifiers precede the noun. The relative clause comes first (announced by `rena`), then adjectives, then the noun. Each layer adds to the description, building toward the final reveal of what is being described.
