# Correlative constructions

Simple coordination handles most needs, but sometimes you want to emphasize the relationship between connected elements. Correlative constructions do this through parallel repetition of the conjunction, creating a balanced structure that draws attention to the pairing itself.

## Both...and: `nela...nela`

Repeating `nela` before each element creates emphasis on completeness, the sense that both parts matter equally:

```
mia nela shea nela lothea phaelo
1SG COORD peace COORD love feel
(i feel both peace and love.)
```

```
nela sorae nela pheralu welao nai
COORD sun COORD rain good be
(both sun and rain are good.)
```

```
nela mia theo nela thia sheluo
COORD 1SG read COORD 2SG listen
(both i read and you listen.)
```

The repeated `nela` signals that this isn't a casual list. The speaker is deliberately pairing these elements, insisting that each carries equal weight.

## Either...or: `sola...sola`

The same principle applies with `sola`. Repeating it before each alternative emphasizes that a genuine choice is being presented:

```
mia sola theo sola thalo
1SG DISJ read DISJ walk
(i either read or walk.)
```

```
thia sola mia naphe sola mia so wepu
2SG DISJ 1SG help DISJ 1SG FUT go
(either you help me or i will go.)
```

Where single `sola` casually mentions alternatives, `sola...sola` frames the choice as deliberate and significant.

## Neither...nor: `sola...sola` with negation

To express "neither...nor," use the correlative `sola...sola` structure with the negation particle `ma` on the verb:

```
mia sola theo sola sheluo ma wilao
1SG DISJ read DISJ listen NEG long-for
(i want neither to read nor to listen.)
```

```
sola sorae sola pheralu ma nai
DISJ sun DISJ rain NEG be
(there is neither sun nor rain.)
```

## Not...but: `ma...thona`

This construction corrects or redirects, rejecting one characterization in favor of another:

```
ma shaelo thona thesua
NEG brave BUT mindful
(not brave but mindful.)
```

```
shia ma haolu thona sheluo
3SG NEG speak BUT listen
(ko does not speak but listens.)
```

```
mia ma phelora peloru thona phelora shiro nila
1SG NEG beautiful flower BUT beautiful tree see
(i see not a beautiful flower but a beautiful tree.)
```

The `ma...thona` pattern is particularly useful in Phi's collaborative dialogue style. Rather than flatly contradicting someone, a speaker can acknowledge what's being rejected while redirecting toward what they actually mean. It corrects without dismissing.

## When to use correlatives

Correlative constructions carry more weight than simple conjunction. Use them when:

- You want to emphasize that both elements matter equally (nela...nela)
- You're presenting a genuine, significant choice (sola...sola)
- You need to explicitly exclude all options (sola...sola + ma)
- You're correcting a misunderstanding or redirecting attention (ma...thona)

For everyday connection without special emphasis, the simple conjunctions from the previous sections work fine. Correlatives are a tool for precision, not a requirement.
