# The relativizer `rena`

**rena** opens a clause that modifies a following noun or, when the head is implicit, occupies a noun-phrase position by itself.

## Saying `rena`

Stress the first syllable. Tap or briefly trill **r** at the alveolar ridge and release into **e**. For dental **n**, place the tongue tip at the backs of the front teeth, then open into **a**: /ˈre̞.n̪ä/. The movement is a pronunciation path, not a second grammar hidden inside the word.

## Headed structure

A headed relative has this shape:

```
[rena CLAUSE] NOUN
```

Everything after **rena** and before the head noun belongs to the description.

```
rena nophi kealo miona ha nai.
REL story create person PROX be.
(The person who creates stories is here.)
```

```
mia rena mia to theo shelu lothea.
1SG REL 1SG PST read book love.
(I love the book that I read.)
```

```
rena mua shelira nai ruela mioru nai.
REL LOC forest be path beautiful be.
(The path that is in the forest is beautiful.)
```

## Why there is no closer

The complement frames **mena ... meno**, **wela ... welo**, and **shola ... sholo** need an explicit closing word before the matrix predicate. A headed **rena** clause ends at its head noun. In **rena mia to theo shelu**, the relative predicate **theo** completes before **shelu** arrives as the book that was read.

Headless relatives still keep **rena**, but context supplies the missing head. Their noun-phrase position marks where the construction ends; Chapter 20 closes with that pattern.

## The gap

The head noun is absent from one role inside the relative clause. A schematic gap makes the relation visible:

```
rena mia ___ to theo shelu.
REL 1SG [gap] PST read book.
"the book that I read" (noun phrase).
```

The gap occupies object position. In the next phrase it occupies subject position:

```
rena ___ nophi kealo miona.
REL [gap] story create person.
"the person who creates stories" (noun phrase).
```

**rena** does not change form according to the missing role.

If a preposition governs the gap, the preposition remains where Phi normally places it:

```
mia rena mia mua to thalo shelira lothea.
1SG REL 1SG LOC PST walk forest love.
(I love the forest that I walked in.)
```

Inside the relative clause, **mua** still precedes its missing object. The head **shelira** supplies that object after the clause, so the preposition neither disappears nor moves to the end.
