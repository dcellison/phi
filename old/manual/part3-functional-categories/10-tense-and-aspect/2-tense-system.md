# Tense System

## The Three-Way Temporal Reference Framework

Phi's **tense system** provides speakers with precise temporal reference through three systematic particles that establish when events occur relative to the speech moment. This system balances simplicity with expressiveness, enabling clear temporal marking across all communicative contexts.

## Core Tense Particles

### Complete Temporal Reference System

| Particle | Function | Temporal Relation | Speech Time Reference |
|----------|----------|-------------------|----------------------|
| `li` | Past | Before speech time | Event completed prior to now |
| `ta` | Present | At speech time | Event occurring now or habitually |
| `su` | Future | After speech time | Event expected after now |

### Basic Temporal Patterns

**Past reference** with `li`:
```
mia li whera.
1SG PST learn
"I learned." (completed past action)

mia li na he nowhea whelu.
1SG PST OBJ HUM person love
"I loved the person." (past emotional state)
```

**Present reference** with `ta`:
```
mia ta whera.
1SG PRS learn
"I learn." (current action or general truth)

mia ta na he nowhea whelu.
1SG PRS OBJ HUM person love
"I love the person." (current emotional state)
```

**Future reference** with `su`:
```
mia su whera.
1SG FUT learn
"I will learn." (planned future action)

mia su na he nowhea whelu.
1SG FUT OBJ HUM person love
"I will love the person." (future emotional commitment)
```

## Cross-Linguistic Perspective

### Universal Tense Phenomena

**Most languages distinguish past, present, and future**, though with varying complexity:

**English**: Past (-ed), present (∅), future (will/shall)
- Complex auxiliary system
- Irregular verb forms
- Tense-aspect fusion

**Spanish**: Rich inflectional system with person marking
- canté (1sg.past) / canto (1sg.present) / cantaré (1sg.future)
- Person-number-tense fusion

**Mandarin**: Aspect-prominent with optional tense marking
- 了 (le) for completion
- 在 (zài) for ongoing action
- 会 (huì) for future possibility

### Phi's Systematic Advantages

**Clear particle system**: No morphological complexity
```
English: learn/learned/will learn (3 different forms)
Phi:     whera with li/ta/su   (systematic particle marking)
```

**Consistent marking**: All verbs work identically
```
whera (learn): li whera / ta whera / su whera
phera (be):    li phera / ta phera / su phera
sharo (go):    li sharo / ta sharo / su sharo
```

**Separation from person**: Tense independent of subject
```
mia li whera.    "I learned."
sha li whera.    "S/he learned."
lo nowhea li whera. "People learned."
```

## Temporal Reference Semantics

### Past Tense (`li`) - Completed Prior Events

**Completed actions**:
```
mia li na ne whethea whera.
1SG PST OBJ INAN book learn
"I learned from books." (completed learning event)
```

**Past states**:
```
mia li haphe phera.
1SG PST happy be
"I was happy." (past emotional state)
```

**Past habits** (when combined with habitual aspect):
```
mia li po whera.
1SG PST HAB learn
"I used to learn." (past habitual action)
```

### Present Tense (`ta`) - Current Time Reference

**Ongoing actions**:
```
mia ta whera.
1SG PRS learn
"I learn." / "I am learning."
```

**Current states**:
```
mia ta haphe phera.
1SG PRS happy be
"I am happy." (current emotional state)
```

**General truths**:
```
ne whethea ta riphe phera.
INAN book PRS important be
"Books are important." (general truth)
```

**Habitual actions**:
```
mia ta po whera.
1SG PRS HAB learn
"I usually learn." (current habitual pattern)
```

### Future Tense (`su`) - Projected Events

**Planned actions**:
```
mia su whera.
1SG FUT learn
"I will learn." (intentional future action)
```

**Predicted states**:
```
mia su haphe phera.
1SG FUT happy be
"I will be happy." (predicted future state)
```

**Future obligations**:
```
mia su ru whera.
1SG FUT OBLG learn
"I will need to learn." (future necessity)
```

## Tense-Evidentiality Interactions

### Compatible Tense-Evidence Combinations

**Past + Memory evidential**:
```
mu mia li whera.
MEM 1SG PST learn
"I learned." (I remember this past event)
```

**Present + Direct evidential**:
```
hi mia ta whera.
DIR 1SG PRS learn
"I learn." (I directly observe current learning)
```

**Future + Inferential evidential**:
```
ro mia su whera.
INF 1SG FUT learn
"I will learn." (I infer this future development)
```

### Problematic Combinations

**Direct evidential with future** (cannot directly observe future):
```
❌ *hi mia su whera.
    DIR 1SG FUT learn
    (Cannot directly observe future events)

✅ ro mia su whera.
   INF 1SG FUT learn
   "I will learn." (inference-based future)
```

**Memory evidential with present** (memories are of past events):
```
? mu mia ta whera.
  MEM 1SG PRS learn
  "I learn." (remembering current events - marked)

✅ mu mia li whera.
   MEM 1SG PST learn
   "I learned." (remembering past events - natural)
```

## Register and Context Effects

### Formal Academic Contexts

**Explicit temporal marking preferred**:
```
Formal:
so hi ne moruphea li ni riphe ta phera.
POL DIR INAN findings PST PRF important PRS be
"The findings have politely proven important." (explicit past + present)

Casual:
ne moruphea riphe ta phera.
INAN findings important PRS be
"The findings are important." (implicit temporal reference)
```

**Future predictions in academic writing**:
```
ro ne reshathi su mo ma naphe ta phera.
INF INAN research FUT CMPR EMPH precise PRS be
"Research will be *more* precise." (inferential future prediction)
```

### Narrative Contexts

**Past tense dominance in stories**:
```
Narrative sequence:
mia li sharo. mia li na he nowhea whelu. mia li haphe phera.
1SG PST go. 1SG PST OBJ HUM person love. 1SG PST happy be
"I went. I loved the person. I was happy."
```

**Present tense for immediate relevance**:
```
mia li whera mi mia ta ni whera.
1SG PST learn CNTR 1SG PRS PRF learn
"I learned, but I have (now) learned." (past action with present relevance)
```

### Conversational Contexts

**Flexible temporal marking**:
```
Minimal:
mia whera.
1SG learn
"I learn." (context provides temporal reference)

Explicit:
mia ta whera.
1SG PRS learn
"I (am currently) learning." (explicit present marking)
```

## Advanced Temporal Constructions

### Sequential Temporal Reference

**Multi-event temporal sequences**:
```
mia li whera. mia ta ni whera. mia su mo whera.
1SG PST learn. 1SG PRS PRF learn. 1SG FUT CMPR learn
"I learned. I have learned. I will learn more."
```

### Complex Temporal Relationships

**Cross-temporal comparisons**:
```
mia li le whera mi mia ta mo whera.
1SG PST LESS learn CNTR 1SG PRS CMPR learn
"I learned less, but I learn more (now)."
```

**Temporal-modal integration**:
```
mia li ru whera mi me li whera.
1SG PST OBLG learn CNTR NEG PST learn
"I should have learned, but I didn't learn."
```

## Processing and Acquisition

### Cognitive Advantages

**Transparent temporal mapping**: Each tense particle clearly maps to temporal relationship
**Systematic application**: All verbs follow identical patterns
**Compositional semantics**: Tense meaning combines predictably with other particles

### Learning Sequence

**Stage 1**: Master basic temporal distinctions
- `li` for completed past events
- `ta` for current situations  
- `su` for future plans

**Stage 2**: Learn tense-evidentiality compatibility
- Which evidentials work with which tenses
- Common vs. marked combinations

**Stage 3**: Integrate with aspect and modality
- How tense combines with aspectual distinctions
- Temporal-modal interactions

**Stage 4**: Develop register sensitivity
- Formal vs. casual temporal marking patterns
- Context-appropriate tense usage
