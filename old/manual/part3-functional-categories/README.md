## The Five Functional Categories

### 9. Evidentiality System ✅
**"Knowledge and Information Source"**

The evidentiality system manages **epistemic relationships** between speakers and the information they convey. Phi's six-way evidential distinction enables precise marking of knowledge sources and epistemic stance.

**Core particles**: `hi` (direct), `ro` (inference), `mu` (memory), `pe` (hearsay), `ke` (knowledge), `no` (general)

**Key innovations**:
- Six-way evidential distinction (unusually rich system)
- Obligatory evidential marking in formal contexts
- Systematic evidential-tense interactions
- Epistemic precision for academic and technical communication

### 10. Tense and Aspect
**"Temporal Reference and Aspectual Distinctions"**

The temporal system provides **comprehensive time-aspect marking** for precise temporal reference. Phi combines simple tense with rich aspectual distinctions to enable nuanced temporal expression.

**Core particles**: 
- **Tense**: `li` (past), `ta` (present), `su` (future)
- **Aspect**: `we` (desiderative), `la` (progressive), `ni` (perfect), `po` (habitual), `pu` (perfective), `ri` (imperfective), `wi` (inceptive), `wu` (cessative)

**Key innovations**:
- Clear tense-aspect separation in syntax
- Rich aspectual system with eight distinctions
- Systematic temporal-evidential interactions
- Narrative-friendly aspectual marking

### 11. Politeness and Social Register
**"Social Deixis and Register Variation"**

The politeness system manages **social relationships** and register appropriateness in communication. Phi's streamlined politeness system provides essential social marking without excessive complexity.

**Core particles**: `so` (politeness), plus register-sensitive particle density patterns

**Key innovations**:
- Single, versatile politeness particle
- Register-based particle density variation
- Integration with evidentiality for social appropriateness
- Context-sensitive application patterns

### 12. Discourse Management
**"Topic, Contrast, and Information Structure"**

The discourse system manages **information flow** and discourse coherence across utterances. Phi provides systematic tools for topic establishment, contrast marking, and information structure management.

**Core particles**: `ha` (topic), `mi` (contrast), plus information structure effects from other slots

**Key innovations**:
- Clear topic-contrast distinction
- Integration with three-slot architecture
- Cross-slot information structure effects
- Discourse-level particle scope interactions

### 13. Comparison and Quantification
**"Degree, Quantity, and Comparative Constructions"**

The comparison system enables **precise degree and quantity marking** for descriptive and analytical expression. Phi's systematic comparison particles support nuanced qualitative and quantitative distinctions.

**Core particles**: 
- **Comparison**: `pa` (most/superlative), `mo` (more/comparative), `sa` (equal), `le` (less), `re` (least)
- **Number**: `wo` (paucal 2-5), `lo` (general plural), `no` (greater plural 6+)

**Key innovations**:
- Five-way comparison system
- Three-way number distinctions
- Emphasis-comparison interactions
- Cross-linguistic comparative analysis

## Cross-Category Interactions

### Systematic Integration

Functional categories don't operate in isolation. Key interaction patterns:

**Evidentiality + Temporal**: Different evidentials license different temporal references
```
✅ hi mia ta whera.    (direct evidence + present - natural)
✅ mu mia li whera.    (memory evidence + past - natural)
❌ hi mia su whera.    (direct evidence + future - problematic)
```

**Politeness + Discourse**: Social register affects information structure choices
```
Formal:   so ha ne whethea, mia ta whera.   (polite + topic-prominent)
Informal: mia ta ma whethea whera.          (emphasis-prominent)
```

**Comparison + Emphasis**: Degree marking integrates with focal prominence
```
pa ma riphe    (superlative + emphasis = "*most* important")
mo ma misha    (comparative + emphasis = "*more* beautiful")
```

### Processing Implications

Cross-category interactions create **coherent semantic profiles**:
- **Academic register**: Inferential evidentials + complex temporal marking + formal politeness
- **Narrative discourse**: Memory evidentials + aspectual sequence + topic management
- **Argumentative style**: Emphasized comparisons + contrastive discourse + epistemic precision
