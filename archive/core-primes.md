# Tier 1 Aligned NSM Primes for Phi Core Vocabulary

## Overview
This document contains the Natural Semantic Metalanguage (NSM) primes that align with or don't conflict with Phi's Five Pillars of philosophy. Out of 65 total NSM primes, approximately 55-57 align well with Phi's principles.

## Aligned Primes by Category

### Substantives & Determiners
- **I, YOU** - Essential (though need careful implementation to avoid ego-separation)
- **SOMEONE, SOMETHING** - Neutral, necessary
- **THIS** - Neutral, useful for deixis
- **THE SAME** - Supports recognition of unity/similarity

### Quantifiers (all align well)
- **ONE, TWO** - Basic counting
- **SOME, ALL** - Necessary for expression
- **MUCH/MANY, LITTLE/FEW** - Useful for non-specific quantities

### Positive Evaluator
- **GOOD** - Aligns when understood as harmonious/beneficial

### Descriptors (neutral)
- **BIG, SMALL** - Simple size descriptions

### Mental Predicates (strongly align)
- **THINK** - Supports mindfulness
- **KNOW** - Epistemic awareness
- **WANT** - Basic desire (can be mindful)
- **FEEL** - Emotional awareness
- **SEE, HEAR** - Sensory awareness

### Speech (align with conscious communication)
- **SAY** - Basic communication
- **WORDS** - Meta-linguistic awareness
- **TRUE** - Supports honesty/authenticity

### Actions/Events (neutral to positive)
- **DO** - Basic action
- **HAPPEN** - Acknowledges process
- **MOVE** - Change/flow

### Existence (mostly align)
- **THERE IS/EXIST** - Basic existence
- **BE (SOMEWHERE)** - Location/presence
- **LIVE** - Celebrates life

### Time (all align - support mindfulness of temporality)
- **WHEN/TIME** - Temporal awareness
- **NOW** - Present moment awareness
- **BEFORE, AFTER** - Temporal relationships
- **A LONG TIME, A SHORT TIME** - Duration
- **FOR SOME TIME** - Temporal extent
- **MOMENT** - Mindful time units

### Space (all align - support awareness of place)
- **WHERE/PLACE** - Spatial awareness
- **HERE** - Present location
- **ABOVE, BELOW** - Vertical relations
- **FAR, NEAR** - Distance
- **SIDE** - Lateral relations
- **INSIDE** - Containment
- **TOUCH** - Physical connection

### Logical Concepts (mostly align)
- **NOT** - Necessary for negation
- **MAYBE** - Supports uncertainty/humility
- **CAN** - Possibility/empowerment
- **BECAUSE** - Understanding causation
- **IF** - Conditional thinking
- **LIKE~AS** - Similarity/connection
- **KIND OF** - Categories without rigid boundaries
- **PART OF** - Supports interconnectedness

### Augmentor
- **VERY** - Needs mindful implementation

## Problematic Primes Excluded

The following NSM primes were identified as conflicting with Phi's philosophies:

- **BAD** - Creates binary judgment, conflicts with Peace Linguistics and Buddhist non-dualism
- **DON'T WANT** - Negative framing that emphasizes rejection
- **DIE** - May conflict with cyclical/transformative view of life
- **OTHER~ELSE** - Emphasizes separation rather than interconnectedness
- **HAVE** - Problematic possession grammar (needs reframing)

## Implementation Notes

1. These aligned primes form the semantic foundation for Phi's core vocabulary
2. Each prime should be implemented with careful attention to sound-meaning correspondence
3. Compound concepts should be built compositionally from these primes in ways that reflect Phi's values
4. The excluded primes may need alternative expressions through compositional means
