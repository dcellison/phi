# Retired writing-system studies

This directory preserves Phi's native-glyph exploration and recovered historical writing material. The native project was retired on 23 July 2026. Its Maya-inspired blocks, featural-alphasyllabary studies, and pen or quill sketches are records of an abandoned direction, not a third Phi writing mode and not a promise of later work.

| Path | What it preserves |
|---|---|
| [`glyph_mode/`](glyph_mode/) | The 2026 native-glyph design brief, cited visual references, concept sheets, and word studies. |
| [`mayan_glyph_mode.md`](mayan_glyph_mode.md) | The earlier Maya-inspired block proposal, including grammar and vocabulary that later became obsolete. |
| [`recovered/`](recovered/) | Tengwar and orthography source recovered from the 2021 LaTeX manual, plus its Obsidian CSS snippet. |

Current Phi uses romanization and the [Phi Tengwar mode](../../tengwar/mode.md). The current specification and renderer assets live under [`tengwar/`](../../tengwar/).
