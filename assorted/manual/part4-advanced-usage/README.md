## Advanced Integration Patterns

### Multi-System Coordination

**Triple system integration** (evidential + temporal + modal):
```
Academic Context:
so ke mia li ru ni na he lo ma ruphea te whera.
POL KNOW 1SG PST OBLG PRF OBJ HUM PL EMPH information VRB learn
"I should have learned from *the* information." (knowledge-based evidence + past obligation + perfect aspect + emphasis)

Conversational Context:
mia ru whera.
1SG OBLG learn
"I should learn." (minimal marking, context-dependent)
```

**Quadruple system coordination** (evidential + temporal + modal + discourse):
```
Complex Academic Discourse:
so ro ha ne reshathi, mia li ru la ni mo ma naphe te whera.
POL INF TOP INAN research, 1SG PST OBLG PROG PRF CMPR EMPH precise VRB learn
"As for research, I should have been progressively learning *more* precisely." 
(politeness + inference + topic + past + obligation + progressive + perfect + comparative + emphasis)
```
