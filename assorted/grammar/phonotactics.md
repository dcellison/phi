# phonotactics

> phonotactics refers to the rules governing the permissible combinations of phonemes
> in a language. it defines which sound sequences are allowed to form syllables and
> words, and which combinations are prohibited. these patterns are often unconsciously
> followed by native speakers and give each language its distinctive sound.

## introduction to phi phonotactics

phi phonotactics are characterized by a highly regular and systematic approach to
sound combinations. the language employs a deliberately constrained phonological 
inventory that creates a distinctive airy quality while maintaining ease of 
pronunciation. crucially, phi's phonotactic patterns are directly tied to grammatical 
function, with each part of speech having its own unique sound signature.

this phonological-grammatical correlation serves multiple purposes:
1. It makes words immediately identifiable by their sound pattern
2. It creates a language with an internally consistent and harmonious sound
3. It reinforces phi's core principle that each word serves exactly one grammatical function

## phonological inventory

phi utilizes a carefully selected inventory of sounds that balance simplicity with
sufficient diversity for a rich vocabulary.

### consonants

phi employs nine single consonants:

| consonant | example  | approximate sound                 |
| --------- | -------- | --------------------------------- |
| h         | ha-      | /h/ as in English "hat"           |
| l         | li-      | /l/ as in English "light"         |
| m         | ma-      | /m/ as in English "mat"           |
| n         | na-      | /n/ as in English "net"           |
| p         | pa-      | /p/ as in English "pat"           |
| r         | ra-      | /ɾ/ as in Spanish "pero"          |
| s         | sa-      | /s/ as in English "sat"           |
| t         | ta-      | /t/ as in English "tap"           |
| w         | wa-      | /w/ as in English "wet"           |

### fricative digraphs

phi uses four two-letter fricative combinations that function as single phonological units:

| digraph | example  | approximate sound                 |
| ------- | -------- | --------------------------------- |
| ph      | pha-     | /f/ as in English "fan"           |
| wh      | wha-     | /ʍ/ as in careful "which"         |
| th      | tha-     | /θ/ as in English "thin"          |
| sh      | sha-     | /ʃ/ as in English "ship"          |

### vowels

phi employs five basic vowels:

| vowel    | example  | approximate sound                 |
| -------- | -------- | --------------------------------- |
| i        | -i-      | /i/ as in English "meet"          |
| u        | -u-      | /u/ as in English "boot"          |
| e        | -e-      | /e/ as in Spanish "que"           |
| o        | -o-      | /o/ as in Spanish "no"            |
| a        | -a-      | /a/ as in Spanish "casa"          |

### vowel pairs

vowel pairs in phi consist of any two distinct vowels in sequence, creating vowel 
hiatus (two adjacent vowels pronounced in sequence). identical vowel pairs (ii, uu, 
ee, oo, aa) are prohibited.

allowed vowel pairs (20 total):
- ia, ie, io, iu
- ua, ue, ui, uo
- ea, ei, eu, eo
- oa, oi, ou, oe
- ae, ai, au, ao

## syllable structure

phi syllables follow strict patterns that contribute to the language's distinctive sound.
a syllable in phi must be one of:

1. **Primary syllable structure**: `(C)CV`
   - A consonant or consonant cluster followed by a vowel
   - If a syllable begins with a consonant cluster, it must be one of the fricative digraphs (ph, wh, th, sh)
   - Examples: `li`, `pha`, `the`, `shu`

2. **Secondary syllable structure**: `V`
   - A single vowel
   - Can only occur immediately after a `(C)CV` syllable
   - Results from vowel hiatus in vowel pairs
   - Examples: the second syllable in `phi.a`, `whu.e`, `the.o`

all syllables in phi are open syllables, meaning they end with vowels. there are no 
syllables ending in consonants.

### syllable restrictions

1. **No duplicate syllables**: no word in phi may contain identical syllables
2. **Fricative digraphs only initially**: consonant clusters are only permitted at the beginning of syllables
3. **Open syllables only**: all syllables must end in vowels
4. **V syllables only after (C)CV**: vowel-only syllables occur exclusively after primary structure syllables

## word patterns by part of speech

phi's most distinctive phonotactic feature is that different parts of speech follow 
different phonological patterns. this creates an isomorphic relationship between sound 
and grammatical function.

### nouns: `[C/F][V/P][F][P]`

nouns follow the pattern of consonant/fricative + vowel/vowel pair + fricative + vowel pair. 
this pattern produces words with a soft, airy quality due to the fricatives.

| pattern component | options                                   | example    |
| ----------------- | ----------------------------------------- | ---------- |
| [C/F]             | h, l, m, n, p, r, s, t, w, ph, wh, th, sh | l-, ph-    |
| [V/P]             | i, u, e, o, a, or any vowel pair          | -i-, -ui-  |
| [F]               | ph, wh, th, sh                            | -ph-, -th- |
| [P]               | any vowel pair                            | -ai, -ui   |

examples:
- `haephui` (dune): h + ae + ph + ui
- `phuithui` (mud): ph + ui + th + ui
- `liphai` (tree): l + i + ph + ai
- `thephoa` (person): th + e + ph + oa
- `nuthui` (pebble): n + u + th + ui

### verbs: `[F][V][C][V]`

verbs follow a pattern that always begins with a fricative, followed by a vowel, then a 
consonant, and ending with a vowel. this gives verbs a more utilitarian, direct sound.

| pattern component | options                                   | example    |
| ----------------- | ----------------------------------------- | ---------- |
| [F]               | ph, wh, th, sh                            | ph-, wh-   |
| [V]               | i, u, e, o, a                             | -a-, -o-   |
| [C]               | h, l, m, n, p, r, s, t, w                 | -m-, -l-   |
| [V]               | i, u, e, o, a                             | -o, -a     |

examples:
- `phamo` (like): ph + a + m + o
- `whami` (rain): wh + a + m + i
- `thunu` (finish): th + u + n + u
- `phola` (walk): ph + o + l + a
- `shoru` (set fire): sh + o + r + u

### adjectives: `[C][V][F][V]`

adjectives follow a pattern that starts with a consonant, followed by a vowel, then a 
fricative, and ending with a vowel. this creates a sound distinct from nouns and verbs.

| pattern component | options                                   | example    |
| ----------------- | ----------------------------------------- | ---------- |
| [C]               | h, l, m, n, p, r, s, t, w                 | h-, m-, p- |
| [V]               | i, u, e, o, a                             | -a-, -i-   |
| [F]               | ph, wh, th, sh                            | -ph-, -sh- |
| [V]               | i, u, e, o, a                             | -e, -a     |

examples:
- `hashe` (green): h + a + sh + e
- `lophu` (deep): l + o + ph + u
- `mipha` (beautiful): m + i + ph + a
- `pisha` (soft): p + i + sh + a
- `nophi` (mountainous): n + o + ph + i

### adverbs: `[C][V][C][V][C][V]`

adverbs in phi follow a distinctive three-syllable pattern with no fricative digraphs. 
this longer structure creates an immediately recognizable rhythm.

| pattern component | options                                   | example    |
| ----------------- | ----------------------------------------- | ---------- |
| [C]               | h, l, m, n, p, r, s, t, w                 | h-         |
| [V]               | i, u, e, o, a                             | -a-        |
| [C]               | h, l, m, n, p, r, s, t, w                 | -m-        |
| [V]               | i, u, e, o, a                             | -i-        |
| [C]               | h, l, m, n, p, r, s, t, w                 | -t-        |
| [V]               | i, u, e, o, a                             | -e         |

examples:
- `hamite` (how): h + a + m + i + t + e
- `tapine` (quickly): t + a + p + i + n + e
- `walime` (outside): w + a + l + i + m + e
- `timane` (when): t + i + m + a + n + e

### prepositions: `[F][P]`

prepositions consist of a fricative digraph followed by a vowel pair.

| pattern component | options                                   | example    |
| ----------------- | ----------------------------------------- | ---------- |
| [F]               | ph, wh, th, sh                            | ph-, th-   |
| [P]               | any vowel pair                            | -ae, -ue   |

examples:
- `phae` (about): ph + ae
- `thue` (of): th + ue
- `shio` (from): sh + io
- `wheo` (at): wh + eo

### determiners: `[F][P][C][V]`

determiners begin with a fricative digraph, followed by a vowel pair, a consonant, and ending with a vowel.

| pattern component | options                                   | example    |
| ----------------- | ----------------------------------------- | ---------- |
| [F]               | ph, wh, th, sh                            | ph-, th-   |
| [P]               | any vowel pair                            | -ia-, -ue- |
| [C]               | h, l, m, n, p, r, s, t, w                 | -t-, -n-   |
| [V]               | i, u, e, o, a                             | -o, -a     |

examples:
- `phiato` (this): ph + ia + t + o
- `thueta` (that): th + ue + t + a
- `whieso` (which): wh + ie + s + o
- `shoata` (all): sh + oa + t + a

### classifiers: `[C][P]`

classifiers consist of a single consonant followed by a vowel pair.

| pattern component | options                                   | example    |
| ----------------- | ----------------------------------------- | ---------- |
| [C]               | h, l, m, n, p, r, s, t, w                 | t-, l-     |
| [P]               | any vowel pair                            | -eo, -ea   |

examples:
- `teo` (round objects): t + eo
- `lea` (long objects): l + ea
- `moi` (flat objects): m + oi
- `hau` (humans): h + au

### conjunctions: `[C][V][C][V]`

conjunctions follow a simple two-syllable pattern with no fricatives.

| pattern component | options                                   | example    |
| ----------------- | ----------------------------------------- | ---------- |
| [C]               | h, l, m, n, p, r, s, t, w                 | n-         |
| [V]               | i, u, e, o, a                             | -e-        |
| [C]               | h, l, m, n, p, r, s, t, w                 | -n-        |
| [V]               | i, u, e, o, a                             | -e         |

examples:
- `nene` (and): n + e + n + e
- `tupo` (but): t + u + p + o
- `wetu` (if): w + e + t + u
- `renu` (because): r + e + n + u

### interjections: `[C][V][C][P]`

interjections consist of a consonant-vowel syllable followed by a consonant and vowel pair.

| pattern component | options                                   | example    |
| ----------------- | ----------------------------------------- | ---------- |
| [C]               | h, l, m, n, p, r, s, t, w                 | m-         |
| [V]               | i, u, e, o, a                             | -i-        |
| [C]               | h, l, m, n, p, r, s, t, w                 | -m-        |
| [P]               | any vowel pair                            | -ia, -ei   |

examples:
- `mimia` (oh): m + i + m + ia
- `hihei` (hey): h + i + h + ei
- `wowou` (wow): w + o + w + ou
- `tatau` (alas): t + a + t + au

### numbers

phi numbers follow two distinct patterns:

#### digits: `[F][V]`

digits consist of a fricative digraph followed by a vowel.

| pattern component | options                                   | example    |
| ----------------- | ----------------------------------------- | ---------- |
| [F]               | ph, wh, th, sh                            | ph-        |
| [V]               | i, u, e, o, a                             | -i         |

examples:
- `phi` (one): ph + i
- `whu` (two): wh + u
- `the` (three): th + e
- `pho` (four): ph + o
- `wha` (five): wh + a

#### magnitudes: `[F][V][F][V]`

magnitudes follow a two-syllable pattern of fricative + vowel + fricative + vowel.

| pattern component | options                                   | example    |
| ----------------- | ----------------------------------------- | ---------- |
| [F]               | ph, wh, th, sh                            | ph-        |
| [V]               | i, u, e, o, a                             | -i-        |
| [F]               | ph, wh, th, sh                            | -th-       |
| [V]               | i, u, e, o, a                             | -a         |

examples:
- `phitha` (ten): ph + i + th + a
- `shupho` (hundred): sh + u + ph + o
- `thuphi` (thousand): th + u + ph + i

### particles: `[C][V]`

particles follow a minimal consonant-vowel pattern.

| pattern component | options                                   | example    |
| ----------------- | ----------------------------------------- | ---------- |
| [C]               | h, l, m, n, p, r, s, t, w                 | l-, m-     |
| [V]               | i, u, e, o, a                             | -i, -a     |

examples:
- `li` (past tense): l + i
- `me` (negation): m + e
- `na` (object marker): n + a
- `lu` (relative clause): l + u

## phonotactic constraints across categories

across all word categories, phi enforces these general constraints:

1. **No duplicate syllables** within a single word
2. **No identical vowel pairs** (ii, uu, ee, oo, aa)
3. **All syllables are open** (end with vowels)
4. **Consonant clusters are restricted** to the fricative digraphs only
5. **Each word follows exactly one** phonotactic pattern corresponding to its grammatical function

the systematic nature of phi's phonotactics makes the language immediately recognizable 
and ensures that words are pronounceable while maintaining their distinctive character. 
the strict correlation between sound patterns and grammatical function serves as a 
fundamental organizing principle of the language.
